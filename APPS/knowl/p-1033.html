First, let's rearrange and factor:

\begin{equation*} 2y^2=xy-x-y+1 = (x-1)(y-1) \end{equation*}

and since \(\gcd(y,y-1)=1\text{,}\) we know \(y-1\) divides \(2\text{,}\) thus \(y-1=-1,1,-2\text{,}\) or \(2\text{,}\) that is, \(y=0,2,-1\text{,}\) or \(3\text{.}\) We can now just try al these values out to see what \(x\) should be:

  • invalidlabel.

    In this case, \(0=(x-1)(0-1)=1-x\text{,}\) i.e. \(x=1\text{.}\)

  • invalidlabel.

    In this case, \(2=2y^2=(x-1)(y-1)=-2(x-1)\text{,}\) so \(x=0\text{.}\)

  • invalidlabel.

    We see that \(8=2y^2 = (x-1)(y-1)=x-1\text{,}\) and so \(x=9\text{,}\) and the solution in this case is \((x,y)=(9,2)\text{.}\)

  • invalidlabel.

    We see that \(18=2y^2 = (x-1)(y-1)=2(x-1)\text{,}\) and so \(x=10\text{,}\) and the solution in this case is \((x,y)=(10,3)\text{.}\)
