We need to show that for any \(\epsilon>0\) there is \(N \in \R\) so that \(n> N\) implies \(|\sqrt{x_{n}}-\sqrt{x}|\lt \epsilon\text{.}\)

Let \(\epsilon>0\text{.}\) Since \(x_n\rightarrow x\text{,}\) there is \(N\) so that \(n> N\) implies \(|x_{n}-x|\lt \epsilon\sqrt{x}\text{,}\) and we take this \(N\text{.}\) We then have, for \(n > N\text{,}\)

\begin{equation*} |\sqrt{x_{n}}-\sqrt{x}| =\left|(\sqrt{x_{n}}-\sqrt{x})\cdot \frac{\sqrt{x_{n}}+\sqrt{x}}{\sqrt{x_{n}}+\sqrt{x}}\right| =\left|\frac{x_{n}-x}{\sqrt{x_{n}}+\sqrt{x}}\right| \end{equation*}
\begin{equation*} =\frac{|x_{n}-x|}{\sqrt{x_{n}}+\sqrt{x}} \leq \frac{|x_{n}-x|}{\sqrt{x}} \lt \frac{\epsilon \sqrt{x}}{\sqrt{x}} = \epsilon\text{.} \end{equation*}
