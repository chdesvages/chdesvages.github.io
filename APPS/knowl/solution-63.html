Solution 1.

We need to find the roots of \(x^3+1=0\text{,}\) which are solutions to \(x^3=-1=e^{\pi i}\text{.}\) One solution is clearly \(1\text{.}\) Another we can get by taking 1/3 the exponent of \(e^{\pi i}\text{,}\) which is \(w=e^{\pi i/3} = \frac{1}{2}+\frac{\sqrt{3}}{2}\text{.}\) We know that roots of real polynomials come in conjugate pairs, and so the other root is \(\overline{w}\text{.}\) Thus,

\begin{align*} x^3+1 \amp = (x-(-1))(x-w)(x-\overline{w}) = (x+1)(x^2-wx-\overline{w}x+\overline{w}w)\\ \amp = (x+1)(x^2-x+|w|^2) = (x+1)(x^2-x+1)\text{.} \end{align*}
