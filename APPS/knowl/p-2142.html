So the first cycle is \((1 \; 3 \; 5 \; 4 \; 2)\text{.}\) Note that all numbers in \(\{1,2,...,5\}\) have been used, and so \(fg = (1 \; 3 \; 5 \; 4 \; 2)\text{.}\) Now we do the same for \(gf\text{,}\) starting with \(1\) again:

\begin{gather*} gf(1) = g(f(1)) = g(2) = 3,\\ gf(3) = g(f(3)) = g(1) = 2,\\ gf(2) = g(f(2)) = g(3) = 4,\\ gf(4) = g(f(4)) = g(5) = 5,\\ gf(5) = g(f(5)) = g(4) = 1 \end{gather*}

and since this uses up all numbers from 1 to 5, we get \(gf = (1\; 3 \; 2 \; 4 \; 5)\neq fg\text{.}\)
