Proof

Suppose \(p_1^{r_{1}}\cdots p_k^{r_{k}} = n = q_1^{s_{1}}\cdots q_\ell^{s_{\ell}}\) are two decompositions. By cancelling any common factors, we can assume that no \(p_i\) equals any \(q_j\text{.}\) If there are any \(p_i's\) and \(q_j's\) remaining, CorollaryÂ 8.3.5 implies each \(p_{i}\) equals some \(q_j\text{,}\) which is a contradiction, thus there can be no terms remaining, so the two factorizations must have been equal.
