Proof

The proof of this uses the so-called “zig-zag” trick: Write all the rational numbers in an infinite array, and then start listing all the rationals by going up and down each diagonal as follows:

Let us add draw arrows in this infinite matrix as follows

So the list would be:

\begin{equation*} \frac{1}{1}, \;\; \frac{2}{1}, \;\; \frac{1}{2}, \;\; \frac{1}{3}, \;\; \frac{2}{2}, \;\; \frac{3}{1}, \;\; \frac{4}{1}, \;\; \frac{3}{2}, \;\; \frac{2}{3}, \;\; \frac{1}{4}, \;\; \frac{1}{5}, \;\; \frac{2}{4},...\cdots \end{equation*}

Note that this list repeats numbers (e.g. the number 1 appears as both \(\frac{1}{1}\) and \(\frac{2}{2}\)). But we can just remove numbers that repeat, so the modified list looks like this:

\begin{equation*} \frac{1}{1}, \;\; \frac{2}{1}, \;\; \frac{1}{2}, \;\; \frac{1}{3}, \;\; \frac{3}{1}, \;\; \frac{4}{1}, \;\; \frac{3}{2}, \;\; \frac{2}{3}, \;\; \frac{1}{4}, \;\; \frac{1}{5},...\cdots \end{equation*}

and now this gives a bijection from \(\mathbb{N}\) to \(\mathbb{Q}\text{,}\) where we send \(n\in\mathbb{N}\) to the \(n\)th term in this list.
