Solution [STRUCT].0.1.

First note that if \((x,y)\) is a solution, so is \((-x,y)\text{,}\) so we can assume first that \(x\geq 0\text{.}\)Note that \(x^2=16y^2+8y+2\) implies

\begin{equation*} x^2-1 =16y^2+8y+1=(4y+1)^2 \end{equation*}

Note that this implies \(x\neq 0\text{,}\) thus \(x>0\text{.}\) Note that \(x^2-1=(x-1)(x+1)\text{,}\) so their gcd is at most \(2\text{,}\) but \(x^2-1=(4y+1)^2\) which is odd, thus \(x\pm 1\) are coprime. Thus, they are perfect squares that differ by \(2\text{,}\) so \(x+1=m^2\) and \(x-1=n^2\) (since \(x>0\text{,}\) both \(x\pm 1\geq 0\)). We can assume \(m>n\geq 0\text{.}\) But then

\begin{equation*} 2= x+1-(x-1) = m^2-n^2 = (m-n)(m+n)\text{,} \end{equation*}

so we must have that \(m-n=1\) and \(m+n=2\) (since \(m-n\leq m+n\)), which has no integer solutions.
