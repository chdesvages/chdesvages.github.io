Solution 13.5.9.1.

By trying out some primes, we can see that it seems like \(3\) is the only option, so let's try to prove \(p=3\text{.}\) Suppose \(p\neq 3\text{.}\) Note that any prime \(p\neq 3\) satisfies \(p = \pm 1 \mod 3\text{,}\) so \(p^2 = 1 \mod 3\text{.}\) Thus, since \(8 = -1\mod 3\text{,}\)

\begin{equation*} 8p^2+1 = (-1)1+1=0\mod 3\text{,} \end{equation*}

thus \(8p^2+1\) is a prime divisible by 3, which is only possible if \(8p^2+1=3\text{,}\) which is impossible. Thus \(p=3\) is the only solution.
