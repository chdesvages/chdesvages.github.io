The general idea behind overcounting is as follows: suppose you want to count a set of things \(A\text{.}\) Suppose there is another set \(B\) that (a) is easier to count and (b) to each \(a\in A\) there correspond \(m\) elements of \(B\), that is, \(B\) and \(A\) are in \(m\)-to-one correspondence. This means we can pair each element of \(A\) with \(m\) elements of \(B\) so that every element of \(B\) belongs to exactly one pairing. If we call the set of these pairs \(P\text{,}\) then \(m|A|=|P|\) (since every \(a\) is in exactly \(m\) pairongs of the form \((a,b)\in P\)), and \(|P|=|B|\) since every \(b\in B\) belongs in exactly one pairing. Thus, we get that \(m|A|=|B|\text{.}\)
