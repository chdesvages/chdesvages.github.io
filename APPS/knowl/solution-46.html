Solution 1.

If \((x,y)\) was a solution, then \(-y^2 = -2\mod 3\text{,}\)and so \(y^2 = 2\mod 3\text{,}\) but if we look at the squares of integers mod 3, we see that they are just \(0\) and \(1\text{,}\) so there is no integer \(y\) for which \(y^2 = 2 \mod 3\text{,}\) thus there are no solutions to the original equation.
