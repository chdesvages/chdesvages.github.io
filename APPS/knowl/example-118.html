Example 16.1.2.

The set \(\mathrm{S}_{1}\) consists of one bijection \(f:\mathrm{S}_{1}\rightarrow\mathrm{S}_{1}\) defined by \(f(1)=1\text{.}\)

The set \(\mathrm{S}_{2}\) consists of two bijections:

  • a function \(f\) such that \(f(1)=1\) and \(f(2)=2\text{,}\)

  • a function \(g\) such that \(g(1)=2\) and \(g(2)=1\text{.}\)
