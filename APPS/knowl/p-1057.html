Assume that \((x, y, z)\) is a solution in positive integers. Clearly, \(x\) is divisible by \(3\text{,}\) so \(x= 3x_1\) for some positive integer \(x_1\text{.}\) But then \(27x_{1}^{3}+3y^3+9z^3= 0\text{,}\)hence \(9x_1^3+y^3+3z^3= 0\text{.}\) Now \(y= 3y_1\text{,}\) and we find \(3x_1^3+9y^{3}_{1}+z^3= 0\text{.}\) Finally, \(z= 3z_1\) for some positive integer \(z_1\text{,}\) and \(x_{1}^{3}+ 3y_{1}^{3}+ 9z_{1}^{3}= 0\text{.}\)Thus if \((x, y, z)\) is a, integer solution of the equation \(x^3+3y^3+9z^3= 0\text{,}\) then so is \((x/3,y/3,z/3)\text{.}\) Repeating this argument we find that for every positive solution there is a smaller solution in positive integers: but this is nonsense, thus there is no solution in positive integers.
