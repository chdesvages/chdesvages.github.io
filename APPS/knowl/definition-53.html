Definition 15.3.1.

Let \(S\) be set with \(n\) distinct members, and let \(k\) be an integer such that \(0\leq k\leq n\text{.}\) We let

\begin{equation*} {n \choose k} \end{equation*}

denote the number of subsets of \(S\) which have exactly \(k\) members (and the above is read “\(n\) choose \(k\)” because it gives the number of ways we can choose \(k\) members from an \(n\)-element set). (If \(k>n\text{,}\) we set \({n \choose k}=0\)).
