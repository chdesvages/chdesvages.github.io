Proof

Let \(S\) have \(n\) elements. Then \(n2^{n-1}\) is the number of ways to pick an element \(a\in S\) and pair it with a subset \(A\subseteq S\backslash \{a\}\text{.}\) We can also count these pairs by first picking \(k\text{,}\) then picking a subset \(A\) of size \(n-k\) — which can be done \({n\choose n-k}={n\choose k}\) ways — and then picking an element \(a\in S\backslash A\text{,}\) and then adding over all \(k\text{.}\)
