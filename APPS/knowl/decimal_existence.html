Theorem 9.1.2.

Let \(x\in\mathbb{R}\) satisfy \(0 \leq x \lt 1\text{.}\) Then there is a sequence of integers \((a_n)\text{,}\) with \(a_{n}\in \{0,1,\dots,9\}\) for all \(n\text{,}\) so that if we let

\begin{equation*} x_n = \frac{a_1}{10} + \frac{a_2}{100} + \dots + \frac{a_n}{10^n}\text{,} \end{equation*}

then \(x_n \to x\) as \(n \to \infty\text{.}\)
