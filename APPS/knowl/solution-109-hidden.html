We have already seen in the previous exercise that

\begin{equation*} \left(1 + \frac{1}{n}\right)^n \leq 1 + \frac{1}{1!} + \frac{1}{2!} + \cdots + \frac{1}{n!} \end{equation*}

so by the rules for limits we have \(\limn a_n \leq \limn b_n\text{.}\) (Note that \(\lim_n b_n\) exists by our discussion of the number \(e\text{.}\)) To show that \(\limn b_n \leq \limn a_n\) we must control \(1 + \frac{1}{1!} + \frac{1}{2!} + \cdots + \frac{1}{n!}\) by \(\left(1 + \frac{1}{m}\right)^m\) for some \(m\) (which will probably satisfy \(m \gg n\)). Indeed, note that for \(m \geq n\) we have

\begin{equation*} \left(1 + \frac{1}{m}\right)^m = 1 + 1 + \frac{m(m-1)}{2!m^2} + \frac{m(m-1)(m-2)}{3!m^3} + \dots + \end{equation*}
\begin{equation*} + \dots + \frac{m(m-1) \dots (m-n+1)}{n! m^n} + \dots + \frac{m!}{m!m^m} \end{equation*}
\begin{equation*} \geq 1 + 1 + \frac{1}{2!}\frac{m(m-1)}{m^2} + \frac{1}{3!}\frac{m(m-1)(m-2)}{m^3} + \dots + \frac{1}{n!}\frac{m(m-1) \dots (m-n+1)}{m^n}\text{.} \end{equation*}

The hope is that by taking \(m\) sufficiently large we should be able to take all of the terms

\begin{equation*} \frac{m(m-1)}{m^2}, \frac{m(m-1)(m-2)}{m^3}, \dots, \frac{m(m-1) \dots (m-n+1)}{m^n} \end{equation*}

very close to \(1\text{.}\)

Fix \(n\) and fix \(\epsilon>0\text{.}\) Since \(\frac{m(m-1)}{m^2} \to 1\) as \(m \to \infty\) there is an \(M_1\) such that for \(m > M_1\) we have

\begin{equation*} \frac{m(m-1)}{m^2} > 1- \epsilon\text{.} \end{equation*}

Since \(\frac{m(m-1)(m-2)}{m^3}\to 1\) as \(m \to \infty\) there is an \(M_2\) such that for \(m > M_2\) we have

\begin{equation*} \frac{m(m-1)(m-2)}{m^3} > 1- \epsilon\text{.} \end{equation*}

Continuing in the same way, we eventually obtain an \(M_{n-1}\) such that for \(m > M_{n-1}\) we have

\begin{equation*} \frac{m(m-1) \dots (m-n+1)}{m^n} > 1-\epsilon\text{.} \end{equation*}

So for \(m > M :=\max\{M_1, \dots, M_{n-1}\}\) we have

\begin{equation*} \frac{m(m-1) \dots (m-j+1)}{m^j} >1-\epsilon \end{equation*}

for all \(2\leq j \leq n\text{.}\) Hence, for \(m >M\) we have

\begin{equation*} \left(1 + \frac{1}{m}\right)^m \geq 1 + 1 +(1 - \epsilon)\left( \frac{1}{2!} + \cdots + \frac{1}{n!}\right) \geq (1 - \epsilon)\left(1 + \frac{1}{1!} + \frac{1}{2!} + \cdots + \frac{1}{n!}\right)\text{.} \end{equation*}

Therefore

\begin{equation*} \lim_{m \to \infty} \left(1 + \frac{1}{m}\right)^m \geq (1 - \epsilon) \limn \left(1 + \frac{1}{1!} + \frac{1}{2!} + \cdots + \frac{1}{n!} \right)\text{.} \end{equation*}
