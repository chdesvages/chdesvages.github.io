Suppose \((x,y,z)\) was an integer solution. Notice that \(8z+5\) is odd, so \(x^2+2y^2\) is odd as well, thus \(x^2\) is odd, and so \(x\) is odd. Thus, \(x=2n+1\) for some integer \(n\text{.}\) Plugging that back into the above, we get

\begin{equation*} 8z+5 =x^2+2y^2 = (2n+1)^2 +2y^2 = 4n^2 +4n+1 + 2y^2 \end{equation*}

and so

\begin{equation*} 8z+4 = 4n^2+4n+2y^2\text{.} \end{equation*}
