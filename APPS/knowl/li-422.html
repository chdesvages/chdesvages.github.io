Item b.

Again, we use the multiplication principle as in (a), but while there were \(n\) options for \(a_1\text{,}\) now there are \(n-1\) options for \(a_2\) because, for the \(a_i\) to be distinct, we cannot repeat, so \(a_{2}\neq a_{1}\) and so there are only \(n-1\) options for \(a_2\text{.}\) Similarly, for \(a_{3}\) there are \(n-2\) options for what it can be (since it can't equal \(a_{1}\) or \(a_{2}\)). Continuing in this way, after having picked \(a_{1},\dots,a_{i-1}\text{,}\) there will be \(n-i+1\) remaining options for \(a_i\text{.}\) Thus, the total number of sequences \(a_{1}, \dots ,a_{k}\) of \(k\) distinct elements of \(S\) is \(n(n-1)\cdots (n-k+1)\text{.}\)
