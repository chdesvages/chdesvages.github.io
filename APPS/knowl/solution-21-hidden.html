The problem with this argument is (a), as it asserts that, for every \(a\text{,}\) there is a \(b\) so that \(a\sim b\text{.}\) But not every relation requires every element to be related to something.

A counter example could be the relation on \(\{1,2\}\) where only \(1\sim 1\) (so \(2\not\sim 2\) and \(2\not\sim 1\)). Then it is symmetric and transitive, but not reflexive.
