Solution 9.5.7.1.

  • If \(p_1\neq 2\) then \(p_1>2\) and so \(p_1,p_2,\cdots p_{31}\) are all odd, hence \(p_1^4+p_2^4+\cdots+p_{31}^4\) is odd, so in particular \(30\) does not divide \(p_{1}^{4}+\cdots + p_{n}^{4}\text{,}\) and we get a contradiction. Thus, \(p_{1}=2\text{.}\)

  • If \(p_2\neq 3\) then \(p_{i} = \pm 1\mod 3\) since \(p_{1}=2 = -1\text{,}\) and for \(i\geq 2\text{,}\) \(p_{i}\neq 3\text{.}\) Thus, \(p_{i}^{2} = 1\mod 3\) by FLT, so \(p_i^{4} = 1(\mod 3)\) for \(i=1,2,\cdots,31\text{.}\) Thus,

    \begin{equation*} p_1^4+p_2^4+\cdots+p_{31}^4 = 31 = 1 \mod 3\text{.} \end{equation*}

    Thus, \(p_1^4+p_2^4+\cdots+p_{31}^4\) is not a multiple of \(30\text{.}\) So \(p_3=3\text{.}\)

  • A similar argument shows that \(p_3\) should be \(5\text{:}\) \(p_3\neq 5\) implies \(p_i^4 = 1 (\mod 5)\) for \(i=1,2,\cdots,31\) by FLT and \(p_1^4+p_2^4+\cdots+p_{31}^4 = 1 (\mod 5)\text{.}\)
