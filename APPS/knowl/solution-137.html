Note that \(n-2\) divides \(n^2-4\text{,}\) so The Linear Combination Lemma implies

\begin{equation*} n-2\text{ divides } (n^2-2-(n^2-4))=2\text{.} \end{equation*}

Thus \(n-2\) must be \(\pm 1\) or \(\pm 2\text{,}\) and this is only possible if \(n=1,3\) or \(4\text{.}\)
