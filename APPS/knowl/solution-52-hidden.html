Note that \(p\) is odd so \(p\not = 2,4\mod 6\text{.}\) Also, \(p>3\text{,}\) hence \(p\neq 3\mod 6\text{.}\) Thus, the only remaining possibilities are \(\pm 1 \mod 6\text{.}\) Hence, \(p=6k\pm 1\) for some integer \(k\text{,}\) so that

\begin{equation*} p^2-1 = (6k\pm 1)^2-1=36k^2\pm 12k=12(3k^2\pm k)\text{.} \end{equation*}

Note that \(3k^2\pm k\) is always even (just consider the different cases when \(k\) is odd or even), so in fact \(p^2-1\) is divisible by 24.
