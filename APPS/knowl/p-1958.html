If \(k=0\text{,}\) there is only one subset of size \(0\text{,}\) which is the empty set, and in this case we see that \(\frac{n!}{0!(n-0)!}=\frac{n!}{n!}=1\text{.}\) Now let \(1\leq k\leq n\) and \(S\) be a set of \(n\) elements. Consider the process where we pick an ordered list \((a_{1}, \dots ,a_{k})\) of \(k\) distinct elements from \(S\text{,}\) one at a time, to form a set \(A=\{a_{1}, \dots ,a_{k}\}\) of \(k\) elements. By PropositionÂ 15.1.6, there are \(\frac{n!}{(n-k)!}\) ways of picking the list \((a_{1}, \dots ,a_{k})\text{.}\) However, this is not the same thing as counting the number of subsets of size \(k\text{,}\) since if we picked \(a_1, \dots ,a_k\) in a different order, they would still give rise to the same set \(A=\{a_{1}, \dots ,a_{k}\}\text{.}\) What we have really counted is the number of ways of picking an ordered list of \(k\) elements, not a set of \(k\) elements. The number of orderings of \(\{a_{1}, \dots ,a_{k}\}\) is \(k!\) by the Ordering Theorem. Thus, we have overcounted the number of subsets of size \(k\) by a factor of \(k!\text{,}\) hence if we divide\(\frac{n!}{(n-k)!}\) by \(k!\text{,}\) we get that the number of subsets of size \(k\) is the right side of (15.3.1).
