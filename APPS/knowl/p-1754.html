TheoremÂ 13.1.5 says we can add and multiply modulo \(m\text{.}\) Can we also divide? That is, if \(ad = bd\mod m\text{,}\) do we also have \(a = b\mod m\text{?}\) This is not always the case. Perhaps the simplest example is the following: \(2 = 0 \mod 2\text{,}\) but we cannot divide this equation by \(2\text{,}\) because \(1 \neq 0 \mod 2\text{.}\) More subtly, \(12 = 24 \mod 6\text{,}\) but here we cannot divide by \(3\text{,}\) because \(4 \neq 8 \mod 6\text{.}\) The following theorem says when we can divide in this way.
