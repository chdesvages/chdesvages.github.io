Theorem 9.1.4.

We have the following properties for integers \(a,b,c\in\mathbb{Z}\) and \(m\in\mathbb{N}\text{:}\)

  1. \(a = a\mod m\text{.}\)

  2. If \(a = b\mod m\text{,}\) then \(b = a\mod m\text{.}\)

  3. If \(a = b\mod m\) and \(b = c\mod m\text{,}\) then \(a = c\mod m\text{.}\)
