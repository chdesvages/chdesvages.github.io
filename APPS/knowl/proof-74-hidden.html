It is clear to see that letting \(x=n^3, y=n^2\) for any integer \(n\) solves the equation. So we need to show that this is in fact necessary, not only sufficient.

Let \((x,y)\) be a solution to \(x^2=y^3\text{.}\) Note that if \((x,y)\) is a solution, then so is \((-x,y)\text{,}\) and moreover, \(y\geq 0\) since \(y^3=x^2\geq 0\text{.}\) Thus, we can assume for the moment that \(x,y\geq 0\text{,}\) since if \((x,y)\) is a solution with \(x\lt 0\text{,}\) then \((-x,y)\) will be a positive solution.

Since \(x,y\geq 0\text{,}\) we can look at their prime factorizations. Let us write \(x=p_1^{k_1}\cdots p_n^{k_n}\) and \(y=q_1^{l_1}\cdots q_m^{l_m}\) for the prime factorizations for \(x\) and \(y\) respectively, (so we assume \(p_i\lt p_j\) for \(i\lt j\text{,}\) and \(q_r\lt q_s\) for \(r\lt s\)). Then we have:

\begin{equation*} p_1^{2k_1}\cdots p_n^{2k_n} = x^2=y^3 = q_1^{3l_1}\cdots q_m^{3l_m}\text{.} \end{equation*}

Hence, applying the Fundamental Theorem of Arithmetic to \(x^2\text{,}\) we have \(m=n\text{,}\) \(p_i=q_i\text{,}\) for each \(i\text{,}\) and moreover,

\begin{equation*} 2k_i=3l_i\text{,} \end{equation*}

for each \(i\text{.}\) Since 2 and 3 are coprime, Proposition 10.5 implies that \(k_i\) is divisible by 3 and \(l_i\) is divisible by 2. Let \(k_i'=\frac{k_i}{3}\text{.}\) Let \(n:= p_1^{k_1'}\cdots p_n^{k_n'}\text{.}\) Then \(x=n^3\) and \(y=n^2\text{,}\) as claimed. That is, if \((x,y)\) is a solution with \(x,y\geq 0\text{,}\) then \((x,y)=(n^3,n^2)\) for some \(n\geq 0\text{.}\) Thus, if \((x,y)\) is any solution, then \((x,y)=(\pm n^3,n^2)\) for some \(n\geq 0\text{,}\) or alternatively, \((x,y)=(n^3,n^2)\) for some \(n\in \mathbb{Z}\text{.}\)
