Thus, \({2n\choose n}=|P|\text{.}\) If we look at the set of unordered pairs \(\{A,S\setminus A\}\) where \(|A|=n\text{,}\) then this is exactly half the size of \(P\) since to each undordered pair \(\{A,S\setminus A\}\) there corresponds two ordered pairs \((A,S\setminus A)\) and \((S\setminus A,A)\text{.}\) Hence, \(P\) must be even, and so \({2n \choose n}\) is even.
