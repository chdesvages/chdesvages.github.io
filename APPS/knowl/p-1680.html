If \(b> 0\text{,}\) then \(p\) divides \(24\text{,}\) so \(p=2\) or \(p=3\) and as \(a>b\text{,}\)

\begin{equation*} 24 = p^{a}-p^{b} = (p^{a-b}-1)p^{b} \end{equation*}

so \(p^{b}\) divides \(24\text{.}\)

  1. If \(p=2\text{,}\) then \(b=1\) or \(2\text{,}\) and checking both of these cases turns up no solutions for \(a\text{.}\)

  2. If \(p=3\text{,}\) then \(b=1\text{,}\) and we find that \(p^{a} = 24+3^{1}=27=3^3\) so \(a=3\text{.}\)
