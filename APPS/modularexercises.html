Section 9.5 Exercises

The relevant exercises in Liebeck's book are in Chapters 13 and 14.

Checkpoint 9.5.1.

Determine whether the following equations have integer solutions:

  • \(x^2+y^2=9z+3\text{.}\) Solution.

    If \((x,y,z)\) was a solution, then \(x^2+y^2 = 3\mod 9\text{,}\) but the squares modulo \(9\) are \(0,1,4,0,7,0,0,1\text{,}\) and adding any pair of these gives \(0,1,2,4,5,7,8\) modulo \(9\text{,}\) and none of these are \(3\text{,}\) thus there cannot be any solutions to the original equation.

  • \(3x^2-y^2=-2\text{.}\) Solution.

    If \((x,y)\) was a solution, then \(-y^2 = -2\mod 3\text{,}\)and so \(y^2 = 2\mod 3\text{,}\) but if we look at the squares of integers mod 3, we see that they are just \(0\) and \(1\text{,}\) so there is no integer \(y\) for which \(y^2 = 2 \mod 3\text{,}\) thus there are no solutions to the original equation.

  • \(3x^2+2=y^2+6z^3\text{.}\) Solution.

    If \((x,y,z)\) is a solution, we see then that \(y^2 = 2\mod 3\text{,}\) and we have already seen that this has no solutions.

Checkpoint 9.5.2.

Show that If \(n\in\mathbb{N}\text{,}\) then \(n^5\) and \(n\) have the same last digit (that is, the digit in the \(1\)'s place).

Solution.

FLT says that \(n^{4} = 1 \mod 5\text{,}\) hence \(n^{5} = n \mod 5\text{,}\) and so \(n^{5}-n = 0\mod 5\text{.}\) Also, \(n^5-n=n^4(n-1)\text{,}\) and since either \(n\) or \(n-1\) is even, we have \(n^5-n = 0\mod 2\text{.}\) Thus, we have \(2\) divides \(n^5-n\) and \(5\) divides \(n^5-n\text{,}\) hence \(10\) divides \(n^5-n\text{,}\) so the last digit of \(n^5\) and \(n\) must be the same.

Checkpoint 9.5.3.

Find the last digit of \(7^{7^{7^{7}}}\text{.}\)

Solution.

\(7^2=49 = -1\mod 10\text{,}\) \(7^3 = -7 = 3\mod 10\text{,}\) \(7^4 = 1 \mod 10\text{.}\) Thus, as we take powers of \(7\text{,}\) the 1's digit cycles through 7,9,3, and 1. So we should look at what \(7^{7^7}\mod 4\) is.

Note that \(7 = 3\mod 4\text{,}\) and \(7^2 = 1\mod 4\text{,}\) thus odd powers of \(7\) are equivalent to \(3\mod 4\text{.}\) Since \(7^7\) is odd, \(7^{7^7} = 3\mod 4\text{,}\) and so \(7^{7^{7^{7}}} = 3\mod 10\text{.}\)

Checkpoint 9.5.4.

Suppose \(n\in\mathbb{N}\) satisfies \(n = 3\mod 4\text{.}\) Show that \(n\) has a prime factor \(p\) with \(p = 3\mod 4\text{.}\)

Solution.

Note that \(n\) is not even, so \(n\) has prime factorizaiton \(p_{1}^{r_{1}}\cdots p_{k}^{r_{k}}\) for some primes \(2\lt p_{1}\lt \cdots \lt p_{k}\) and integers \(r_{i}>0\text{.}\) Suppose \(p_{i} = 1 \mod 4\) for all \(i\text{.}\) Then \(p_{i}^{r_{i}} = 1 \mod 4\) as well, and so

\begin{equation*} n=p_{1}^{r_{1}}\cdots p_{k}^{r_{k}} = 1 \mod 4\text{,} \end{equation*}

which is a contradiction. Thus, there is \(i\) so that \(p_{i} = 3\mod 4\text{.}\)

Checkpoint 9.5.5.

Show that there are no integer solutions to \(m^3-n^3=3\) or \(m^3-n^3=4\text{.}\) (Hint: if there were solutions, there would also be a solution \(\mod 7\)).

Solution.

Let us look at what cubes are \(\mod 7\text{:}\)

\begin{align*} 1^3 \amp =1 = 1 \mod 7\\ 2^3 \amp =8 = 1 \mod 7\\ 3^3 \amp =27 = -1 \mod 7\\ 4^3 \amp =64 = 1 \mod 7\\ 5^3 \amp =125 = -1 \mod 7\\ 6^3 \amp = 216 = -1 \mod 7 \end{align*}

Thus, the only values \(m^3-n^3\) can be \(\mod 7\) are \(0,\pm 1\text{,}\) and \(\pm 2\text{,}\) or alternatively, \(0,1,2,5,6\text{.}\) In particular, \(m^3-n^3=3\) and \(m^3-n^3=4\) have no solutions.

Checkpoint 9.5.6.

Show that for all primes \(p>3\text{,}\) \(p = \pm 1\mod 6\text{.}\) Use this to show \(24\) divides \(p^2-1\) for all primes \(p>3\text{.}\)

Solution.

Note that \(p\) is odd so \(p\not = 2,4\mod 6\text{.}\) Also, \(p>3\text{,}\) hence \(p\neq 3\mod 6\text{.}\) Thus, the only remaining possibilities are \(\pm 1 \mod 6\text{.}\) Hence, \(p=6k\pm 1\) for some integer \(k\text{,}\) so that

\begin{equation*} p^2-1 = (6k\pm 1)^2-1=36k^2\pm 12k=12(3k^2\pm k)\text{.} \end{equation*}

Note that \(3k^2\pm k\) is always even (just consider the different cases when \(k\) is odd or even), so in fact \(p^2-1\) is divisible by 24.

Checkpoint 9.5.7.

Show that if \(p_{1}\lt \cdots \lt p_{31}\) are prime and \(30\) divides \(p_{1}^{4}+\cdots + p_{31}^{4}\text{,}\) then \(p_{1}=2\text{,}\) \(p_{2}=3\) and \(p_{3}=5\text{.}\) Hint: Assume the contrary and use FLT.

Solution.

  • If \(p_1\neq 2\) then \(p_1>2\) and so \(p_1,p_2,\cdots p_{31}\) are all odd, hence \(p_1^4+p_2^4+\cdots+p_{31}^4\) is odd, so in particular \(30\) does not divide \(p_{1}^{4}+\cdots + p_{n}^{4}\text{,}\) and we get a contradiction. Thus, \(p_{1}=2\text{.}\)

  • If \(p_2\neq 3\) then \(p_{i} = \pm 1\mod 3\) since \(p_{1}=2 = -1\text{,}\) and for \(i\geq 2\text{,}\) \(p_{i}\neq 3\text{.}\) Thus, \(p_{i}^{2} = 1\mod 3\) by FLT, so \(p_i^{4} = 1(\mod 3)\) for \(i=1,2,\cdots,31\text{.}\) Thus,

    \begin{equation*} p_1^4+p_2^4+\cdots+p_{31}^4 = 31 = 1 \mod 3\text{.} \end{equation*}

    Thus, \(p_1^4+p_2^4+\cdots+p_{31}^4\) is not a multiple of \(30\text{.}\) So \(p_3=3\text{.}\)

  • A similar argument shows that \(p_3\) should be \(5\text{:}\) \(p_3\neq 5\) implies \(p_i^4 = 1 (\mod 5)\) for \(i=1,2,\cdots,31\) by FLT and \(p_1^4+p_2^4+\cdots+p_{31}^4 = 1 (\mod 5)\text{.}\)

Checkpoint 9.5.8.

(Regional Mathematical Olympiad 1998, India) Show that if \(5\lt p_{1}\lt \cdots \lt p_{n}\) and \(6\) divides \(p_{1}^{2}+\cdots + p_{n}^{2}\text{,}\) then \(6\) divides \(n\text{.}\)

Solution.

From an earlier exercise, we know that \(p_{i} = \pm 1 \mod 6\text{,}\) hence \(p_{i}^{2} = 1 \mod 6\text{,}\) thus

\begin{equation*} 0=p_{1}^{2}+\cdots + p_{n}^{2} = n\mod 6\text{,} \end{equation*}

where in the first equation we used our problem assumption, thus \(n = 0\mod 6\text{,}\) hence \(6\) divides \(n\text{.}\)

Checkpoint 9.5.9.

Given that \(p\) and \(8p^2+1\) are prime, find \(p\text{.}\)

Solution.

By trying out some primes, we can see that it seems like \(3\) is the only option, so let's try to prove \(p=3\text{.}\) Suppose \(p\neq 3\text{.}\) Note that any prime \(p\neq 3\) satisfies \(p = \pm 1 \mod 3\text{,}\) so \(p^2 = 1 \mod 3\text{.}\) Thus, since \(8 = -1\mod 3\text{,}\)

\begin{equation*} 8p^2+1 = (-1)1+1=0\mod 3\text{,} \end{equation*}

thus \(8p^2+1\) is a prime divisible by 3, which is only possible if \(8p^2+1=3\text{,}\) which is impossible. Thus \(p=3\) is the only solution.

Checkpoint 9.5.10.

Prove that \(42\) divides \(n^7-n\) for all \(n\in\mathbb{N}\text{.}\)

Solution.

Since \(42 = 2\cdot 3 \cdot 7\text{,}\) we just need to verify each of these factors divides \(n^7-n\) for each \(n\in\mathbb{N}\text{.}\)

  • Since \(n^7-n=n^6(n-1)\text{,}\) one of \(n\) and \(n-1\) must be even, so one of \(n^6\) and \(n\) must be even, so \(2\) divides \(n^{7}-n\text{.}\)

  • \(n^7-n = n^4\cdot (n-1)n\cdot (n-1)\text{,}\) and one of \(n-1,n,n+1\) must be divisible by 3.

  • Finally, by FLT,

    \begin{equation*} n^7-n = n-n=0 \mod 7\text{.} \end{equation*}
Checkpoint 9.5.11.

If \(7\) does not divide \(a\text{,}\) then \(7\) divides \(a^3-1\) or \(7\) divides \(a^3+1\text{.}\)

Solution.

Note that since \(7\) does not divide \(a\text{,}\) FLT implies

\begin{equation*} (a^{3}-1)(a^3+1) = a^6-1 = 1-1=0 \mod 7 \end{equation*}

and so \(7\) divides \((a^{3}-1)(a^3+1)\text{,}\) and since \(7\) is prime, it must divide one of the factors.
