Section 4.2 Separating products

In this section we cover some very useful inequalities that allow us to essentially separate terms in a product. They all stem from the following lemma:

Lemma 4.2.1.

For \(u,v\in\mathbb{R}\text{,}\)

\begin{equation*} uv \leq \frac{u^2+v^2}{2}\text{.} \end{equation*}

Notice that on one side of the above inequality, the terms \(u\) and \(v\) are multiplied together, and on the other side, they are separate (but now squared).

Proof.

Since\((u-v)^2\geq 0\text{,}\) we get

\begin{equation*} 0\leq (u-v)^2 =u^2-2uv + v^2\text{.} \end{equation*}

So by (R3), we can add \(2uv\) to both sides to get

\begin{equation*} 2uv \leq u^2+v^2 \end{equation*}

and then by (R4), we can multiply both sides by \(\frac{1}{2}\) to get \(uv \leq \frac{u^2+v^2}{2}\) as desired.

Checkpoint 4.2.2.

When does equality hold in the previous lemma? That is, when do we have \(uv = \frac{u^2+v^2}{2}\text{?}\)

Solution.

Let's retrace the proof of Lemma 4.2.1 backwards trying to use only equalities:

\begin{equation*} uv = \frac{u^2+v^2}{2} \;\;\; \Longleftrightarrow \;\;\; 2uv = u^2+v^2 \;\;\; \Longleftrightarrow \;\;\; 0=u^2+v^2-2uv = (u-v)^{2} \end{equation*}

and this last equality holds if and only if \(u=v\text{.}\)

From this, we get the famous {Arithmetic Mean-Geometric Mean (AM-GM) inequality}:

Theorem 4.2.3. AM-GM Inequality.

Let \(n\in\mathbb{N}\) and \(x_{1},...,x_{n}\geq 0\text{.}\) Then

\begin{equation*} (x_{1}\cdot x_{2}\cdots x_{n})^{\frac{1}{n}}\leq \frac{x_{1}+\cdots + x_{n}}{n}\text{.} \end{equation*}

That is, the geometric mean on the left is at most the arithmetic mean on the right.

Proof.

The \(n=2\) case follows from Lemma 4.2.1: if we set \(u=\sqrt{x_{1}}\) and \(v=\sqrt{x_{2}}\text{,}\) then

\begin{equation} (x_{1}x_{2})^{\frac{1}{2}}=uv\leq \frac{u^2+v^2}{2} = \frac{x_{1}+x_{2}}{2}\text{.}\label{e_x1x2_1_2_x1_x2_2}\tag{4.2.1} \end{equation}

For the \(n>2\) case, see Exercise 5.5.10.

Checkpoint 4.2.4.

For \(x_{1},x_{2}\geq 0\text{,}\) when do we have \((x_{1}x_{2})^{\frac{1}{2}}= \frac{x_{1}+x_{2}}{2}\text{?}\)

Solution.

If there is equality, that means there is equality in (4.2.1), and by Exercise 4.2.2, this holds if and only if \(u=v\text{,}\) that is, if \(\sqrt{x_{1}}=\sqrt{x_{2}}\text{,}\) which holds if and only if \(x_{1}=x_{2}\text{.}\)

The next useful inequality is the Cauchy-Schwarz inequality:

Theorem 4.2.5. Cauchy-Schwarz Inequality.

Let \(n\in\mathbb{N}\text{,}\) \(x_{1},...,x_{n},y_{1},...,y_{n}\in\mathbb{R}\text{.}\) Then

\begin{equation*} x_{1}y_{1}+\cdots + x_{n}y_{n} \leq \sqrt{x_{1}^2+\cdots + x_{n}^{2}} \sqrt{y_{1}^2+\cdots + y_{n}^{2}}\text{.} \end{equation*}

Why is this useful? On the left we have a mixed product of \(x\)-terms and \(y\)-terms, and we are able to relate it to a product of two terms where now the \(x\)'s and \(y\)'s are separated.

We'll just prove the \(n=2\) case for now and return to the general case later. Note that

\begin{equation*} (x_{1}y_{1}+x_{2}y_{2})^2 =x_{1}^2y_{1}^2+x_{2}^2y_{2}^2+2x_{1}y_{1}x_{2}y_{2} \end{equation*}

and using Lemma 4.2.1 on the second term with \(u=x_{1}y_{2}\) and \(v=x_{2}y_{1}\text{,}\) this above is at most

\begin{equation*} \leq x_{1}^2y_{1}^2+x_{2}^2y_{2}^2+2\frac{(x_{1}y_{2})^2+(x_{2}y_{1})^2}{2} =x_{1}^2y_{1}^2+x_{2}^2y_{2}^2+x_{1}^2y_{2}^2+x_{2}^2y_{1}^2 =(x_{1}^2+x_{2}^2)(y_{1}^2+y_{2}^2) \end{equation*}

Thus we have shown

\begin{equation*} (x_{1}y_{1}+x_{2}y_{2})^2\leq (x_{1}^2+x_{2}^2)(y_{1}^2+y_{2}^2) =\left(\sqrt{x_{1}^2+x_{2}^2}\sqrt{y_{1}^2+y_{2}^2}\right)^{2} \end{equation*}

and now by Proposition 4.1.7, we get

\begin{equation*} x_{1}y_{1}+x_{2}y_{2}\leq\sqrt{x_{1}^2+x_{2}^2}\sqrt{y_{1}^2+y_{2}^2}\text{.} \end{equation*}
Example 4.2.6.

Show that

\begin{equation*} \frac{a+b}{2} \leq \left(\frac{a^2+b^2}{2}\right)^{\frac{1}{2}}\text{.} \end{equation*}

Which of our above inequalities should we try using? Lemma 4.2.1 and AM-GM don't seem relevant since we don't have something like \(ab\) on the left, so let's see if we can use the Cauchy-Schwarz inequality: Rewrite \(\frac{a+b}{2}=a\cdot \frac{1}{2} +b\cdot\frac{1}{2}\text{,}\) then the Cauchy Schwarz inequality implies:

\begin{equation*} \frac{a+b}{2} = a\cdot\frac{1}{2} + b\cdot\frac{1}{2} \leq \sqrt{a^2+b^2}\sqrt{\frac{1}{2^{2}}+\frac{1}{2^{2}}} =\left(\frac{a^2+b^2}{2}\right)^{\frac{1}{2}}\text{.} \end{equation*}
