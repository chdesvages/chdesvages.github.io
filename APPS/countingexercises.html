Section 15.7 Exercises

Checkpoint 15.7.1.

How many ways are there for \(n\) people to stand in a circle?

Solution.

Let's first count the number of ways to order people clockwise around a circle where the first person stands at a specific spot. This is just \(n!\text{.}\) However, we are counting the number of ways for \(n\) people to stand in a circle, and so there is no first position, and for such an arrangement, there are \(n\) ways to pick who should be the “first person.” Thus, we overcounted by a factor \(n\text{,}\) so dividing \(n!\) by \(n\) gives the answer \((n-1)!\text{.}\)

Checkpoint 15.7.2.

How many ways are there to partition a set of size \(n\) into two subsets (one of which can be empty)?

Solution.

For 2 sets, if \(S\) is our set, notice that to each \(A\subseteq S\) there corresponds a partition of \(S\text{,}\) namely \(\{A,S\backslash A\}\text{.}\) Thus, the number of partitions is the same as the number of subsets, which is \(2^{n}\text{.}\)

Checkpoint 15.7.3.

Determine how many ways there are to distribute \(9\) sweeties in \(4\) boxes if

  1. The sweeties and boxes are distinct. Solution.

    For each sweetie, there are \(4\) options for which box to place it in, thus by the multiplication principle, the number of options is \(4^{9}\text{.}\)

  2. The sweeties are identical and the boxes are distinct. Solution.

    Using the method in Example 15.3.7, this is equal to the number of ways of placing \(4-1 = 3\) dividers in a line of \(9\) balls, which is \({{9+3}\choose {3}} = {12 \choose 3}\text{.}\)

Checkpoint 15.7.4.

How many numbers at most 200 are divisible by

  1. 5 or 6? Solution.

    The largest multiple of 6 at most 200 is 192, and so there are 192/6=31 multiples of 6 at most 200 (call this set \(A\)). Similarly, there are 200/5=40 multiples of 5 (call this set \(B\)). The largest multiple of \(5\cdot 6=30\) is 180, and so there are at most \(180/30=6\) numbers divisible by 5 and 6, thus

    \begin{equation*} |A\cup B| = |A|+|B|-|A\cap B| = 31+40-6 = 65\text{.} \end{equation*}

  2. 3,4 or 5? Solution.

    Homework problem.

  3. 3,5, or 6? Solution.

    Let \(A,B,C\) be those numbers at most 200 which are divisible by 3,5 and 6 respectively. We want to figure out what \(|A\cup B\cup C|\) is. Notice that \(C\subseteq A\text{,}\) and so \(A\cup B\cup C = A\cup B\text{.}\) Thus, by the inclusion exclusion principle,

    \begin{equation*} |A\cup B\cup C|=| A\cup B|=|A|+|B|-|A\cap B| =66 + 40-13 = 93\text{.} \end{equation*}

Checkpoint 15.7.5.

Use a combinatorial argument to prove that for \(2\leq k\leq n\text{,}\)

\begin{equation*} {k \choose 2} + {n-k \choose 2}+k(n-k) = {n \choose 2}\text{.} \end{equation*}
Solution.

The term on the right is the number of ways to choose two elements from a set \(S\) of size \(n\text{.}\) We can alternatively count this as follows: first let \(2\leq k\leq n\) and let \(A\subseteq S\) have size \(k\text{.}\) Then we can pick a subset of \(S\) of size two either by picking two elements from \(A\) (of which there are \({k \choose 2}\) options), or two elements from \(S\setminus A\) (of which there are \({n-k \choose 2}\) options) , or one from \(A\) and one from \(S\setminus A\) (of which there are \(k(n-k)\) options). Adding these all together gives the left side of the above equation.

Checkpoint 15.7.6.

Use a combinatorial argument to show

\begin{equation*} \binom{nk}{2} = k \binom{n}{2} + n^2 \binom{k}{2}\text{.} \end{equation*}
Solution.

Consider a grid of \(nk\) squares with \(k\) columns and \(n\) rows, and let \(A\) be the set of unordered pairs of distinct squares from the \(nk\) squares in the grid. Then \(|A|=\binom{nk}{2}\text{.}\) Alternatively, we could compute \(|A|\) as follows: let \(A_{1}\) be those pairs of squares that are in different columns and \(A_{2}\) be those pairs of squares in the same column: there are \(\binom{k}{2}\) ways to pick the two columns, then \(n\) ways to pick a square from one column, then another \(n\) ways to pick a square from the second column, thus \(|A_{1}|=n^2 \binom{k}{2}\text{.}\) Next, to count \(A_{2}\text{,}\) there are \(k\) columns to pick from, and then we can pick a subset of size \(2\) in \(\binom{n}{2}\) ways from that column, so \(|A_{2}|=k \binom{n}{2}\text{.}\) Thus,

\begin{equation*} \binom{nk}{2} = |A|=|A_{1}|+|A_{2}|= n^2 \binom{k}{2}+k \binom{n}{2}\text{.} \end{equation*}
Checkpoint 15.7.7.

Give two proofs that \({2n\choose n}\) is even.

Solution. 1

First solution: notice that \({2n\choose n}\) counts the number of sets of size \(n\) from a set \(S\) of size \(2n\text{.}\) Notice that each such set \(A\) of size \(n\) appears once in the first component of the following pairs

\begin{equation*} P=\{(B,S\setminus B) \; | \; B\subseteq S,\;\; |B|=n\}\text{.} \end{equation*}

Thus, \({2n\choose n}=|P|\text{.}\) If we look at the set of unordered pairs \(\{A,S\setminus A\}\) where \(|A|=n\text{,}\) then this is exactly half the size of \(P\) since to each undordered pair \(\{A,S\setminus A\}\) there corresponds two ordered pairs \((A,S\setminus A)\) and \((S\setminus A,A)\text{.}\) Hence, \(P\) must be even, and so \({2n \choose n}\) is even.

Solution. 2

Second solution: for a second proof, by the binomial theorem,

\begin{align*} 2^{2n} \amp = (1+1)^{2n} = \sum_{k=0}^{2n} {2n\choose k} =\sum_{k=0}^{n-1} {2n\choose k}+{2n\choose n}+\sum_{k=n+1}^{2n} {2n\choose k}\\ \amp =\sum_{k=0}^{n-1} {2n\choose k}+{2n\choose n}+\sum_{k=n+1}^{2n} {2n\choose 2n-k}\\ \amp =\sum_{k=0}^{n-1} {2n\choose k}+{2n\choose n}+\sum_{j=0}^{n-1} {2n\choose j}\\ \amp = 2 \sum_{k=0}^{n-1} {2n\choose k}+{2n\choose n} \end{align*}

where in the second line we used the fact that \({m \choose r} = {m \choose m-r}\) and in the third line we made a change of variables \(j = 2n-k\text{.}\) Thus \({2n\choose n}\) is a difference of even numbers and so it must be even.

Checkpoint 15.7.8.

If \(n\geq 5\) distinct objects are arranged in a circle, how many ways are there to choose three of these \(n\) objects so that no two of them are next to each other?

Solution.

Claim: There are \(\frac{1}{6}n(n-4)(n-5)\) ways to choose three of \(n\) distinct objects arranged in a circle so that no two of them are next to each other.

Proof.

Choose one object, which can be done in \(n\) ways. Then the remaining two must be picked from those that aren't adjacent to it, so there are \(n-3\) objects to pick from, (and we can think of these objects as now being arranged along a line). There are \({{n-3} \choose 2}\) ways of choosing any two of these objects and \(n-4\) ways of choosing them to be adjacent. So there are

\begin{equation*} {{n-3} \choose 2} - (n-4) = \frac{(n-3)(n-4)}{2} - (n-4) = \frac{(n-4)(n-5)}{2} \end{equation*}

ways of choosing the remaining two to be non-adjacent. By the multiplication principle, there are \(\frac{n(n-4)(n-5)}{2}\) ways of making the choice, having fixed the first one. But this overcounts by a factor of \(3! = 3\text{,}\) so the final answer is \(\frac{n(n-4)(n-5)}{6}\text{.}\)

Checkpoint 15.7.9.

Give both an algebraic argument and a combinatorial argument to show (for \(0 \leq k \leq n\))

\begin{equation*} k\binom{n}{k} = n\binom{n-1}{k-1} \end{equation*}

in two different ways. {( Hint for the combinatorial proof: For a set \(S\) with \(|S|=n\text{,}\) the left side is the same as counting the number of pairs \((a,A)\) where \(a\in A\subseteq S\) and \(|A|=k\text{.}\) Can you show the right side counts the same set in a different way?)}

Solution.

First we do the algebraic argument:

\begin{equation*} k\binom{n}{k} =\frac{k\cdot n!}{(n-k)!k!} = \frac{n!}{(n-k)!(k-1)!} = \frac{n\cdot (n-1)!}{(n-k)!(k-1)!} \end{equation*}
\begin{equation*} =\frac{n\cdot (n-1)!}{((n-1)-(k-1))!(k-1)!} = n\binom{n-1}{k-1}\text{.} \end{equation*}

Now we give a combinatorial proof. We need to find a common thing these two numbers are counting, or that they count two sets that are in one-to-one correspondence. If you see a term with a product, it might mean that it is counting some set using the Multiplication Principle. In fact, the left side counts the number of ways of pairing a set \(A\) of size \(k\) with an element \(a\in A\text{,}\) i.e. the set of pairs \((a,A)\) where \(A\subseteq S\text{,}\) \(|A|=k\text{,}\) and \(a\in A\text{.}\) Let's see if the right side is in one-to-one correspondence with this set. Note that \(n\) is the number of ways of picking one element \(a\in S\text{,}\) and \(\binom{n-1}{k-1}\) is the number of ways of picking a subset \(B\) of size \(k-1\) from the remaining \(n-1\) elements of \(S\text{,}\) i.e. this counts the set of pairs \((a,B)\) where \(a\in S\text{,}\) \(|B|=k-1\text{,}\) and \(a\not\in B\text{.}\) But each such pair is in one-to-one correspondence with a pair \((a,A)\) (where \(a\in A\) and \(|A|=k\) as before by setting \(A=B\cup \{a\}\text{.}\) Thus, these sets of pairs are in one-to-one correspondence and so they have the same number. This proves the formula.

Checkpoint 15.7.10.

Use a combinatorial argument to show

\begin{equation*} \sum_{k=0}^{n} k{n\choose k} =n2^{n-1}\text{.} \end{equation*}
Solution.
Proof.

Let \(S\) have \(n\) elements. Then \(n2^{n-1}\) is the number of ways to pick an element \(a\in S\) and pair it with a subset \(A\subseteq S\backslash \{a\}\text{.}\) We can also count these pairs by first picking \(k\text{,}\) then picking a subset \(A\) of size \(n-k\) — which can be done \({n\choose n-k}={n\choose k}\) ways — and then picking an element \(a\in S\backslash A\text{,}\) and then adding over all \(k\text{.}\)
