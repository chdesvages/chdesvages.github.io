Section 8.3 Exercises

The only relevant exercise from Liebeck's book is Problem 7 in Chapter 23, which we discussed as Heron's method already.

Checkpoint 8.3.1.

Suppose that \((x_{n})\) is increasing. Show that either \((x_{n})\) converges or else \((x_{n})\) tends to infinity.

Solution.

If \((x_{n})\) is bounded above, then it converges. Otherwise, it is unbounded above, so for all \(M>0\) there is \(N\) so that \(x_{N}\geq M\text{.}\) Since \(x_{n}\) is increasing, we know that for all \(n\geq N\text{,}\)

\begin{equation*} x_{n}\geq x_{N}\geq M\text{.} \end{equation*}

Thus, we have shown that for all \(M\text{,}\) there is \(N\) so that \(n\geq N\) implies \(x_{n}\geq M\text{,}\) which means \(x_{n}\rightarrow\infty\text{.}\)

Checkpoint 8.3.2.

If \((x_n)\) is a bounded sequence and \((x_n+\frac{1}{n})\) is monotonic, does \((x_n)\) converge?

Solution.

Yes. First note that \((x_n + \frac{1}{n})\) is bounded. By the MCT, the bounded monotonic sequence \((x_n+\frac{1}{n})\) converges. Since \(\frac{1}{n}\rightarrow 0\text{,}\) we know that \((x_{n}+\frac{1}{n})-\frac{1}{n}=x_{n}\) converges as well by the rules for limits.

Checkpoint 8.3.3.

Let \((a_{n})\) be a sequence of numbers such that \(0\leq a_{n}\leq 1\) and let \(s_{n}\) denote the product \(a_{1}\cdot a_{2}\cdots a_{n}\text{.}\) Does \((s_{n})\) converge?

Solution.

Yes. Note that as \(a_{n}\geq 0\) for all \(n\text{,}\) \(s_{n}\geq 0\) as well. Moreover, \(s_{n+1} = a_{n+1}s_{n}\leq s_{n}\) since \(a_{n+1}\leq 1\) for all \(n\text{.}\) Thus, \(s_{n}\) is decreasing and bounded below, and hence converges by the MCT.

Checkpoint 8.3.4.

Using the definition of convergence, show that if \(a_n \to L\) as \(n \to \infty\) and if \(b_n = a_{2n}\text{,}\) then \(b_n \to L\) as \(n \to \infty\text{.}\)

Solution.

Let \(\epsilon > 0\text{.}\) There there is an \(N\) such that \(n> N\) implies \(| a_n - L|\lt \epsilon\text{.}\) So \(m > N/2\) implies \(|a_{2m} - L|\lt \epsilon\text{.}\)

Checkpoint 8.3.5.

Let \(a_n = n^{1/n}\text{.}\) Show that \((a_n)\) converges and find its limit. (Hint: First prove that \((1+ \frac{1}{n})^n \leq 4\) for all \(n\text{,}\) and then try to use the MCT. Alternatively, use the formula for \(a^n - b^n\) in [cross-reference to target(s) "x81" missing or not unique] for well-chosen \(a\) and \(b\text{.}\))

Solution.

We can establish the hint using the binomial theorem, see also the discussion of \(e\) in Section 8. We try out whether \((a_n)\) is decreasing. We have

\begin{equation*} a_{n+1} \leq a_n \iff (n+1)^{1/(n+1)} \leq n^{1/n} \iff (n+1)^n \leq n^{n+1} \iff \left(1 + \frac{1}{n}\right)^n \leq n\text{.} \end{equation*}

We saw earlier that \(\left(1 + \frac{1}{n}\right)^n \leq 4\) for all \(n\text{,}\) so we have \(a_{n+1} \leq a_n\) at least when \(n \geq 4\text{.}\) Since \(a_n \geq 1\) we have that \((a_n)\) is bounded below. This is enough for us to deduce from the MCT that it converges. Let \(L = \limn n^{1/n} \geq 1\text{.}\) Let \(b_n = a_{2n} = (2n)^{1/2n} = 2^{1/2n} \sqrt{n^{1/n}}\text{.}\) Since \(b_n\) runs through alternate values of \(a_n\) we have \(b_n \to L\) too. But \(2^{1/2n} \sqrt{n^{1/n}} \to 1^{1/2} \times L^{1/2} = L^{1/2}\) by the rules for limits, Example 6.3 and [cross-reference to target(s) "x2__1_n" missing or not unique]}. Thus \(L = L^{1/2}\) and so \(L = 1\text{.}\)

Alternatively, (assuming \(n\) is even for simplicity)

\begin{equation*} n - 1 = (n^{1/n} - 1)(1 + n^{1/n} + n^{2/n} + \dots + n^{(n-1)/n}) \end{equation*}
\begin{equation*} \geq (n^{1/n} - 1)(n^{n/2n} + n^{(n+2)/2n} + \dots + n^{(2n-2)/2n}) \geq (n^{1/n} - 1) \times \frac{n-1}{2} \times n^{1/2} \end{equation*}

so that

\begin{equation*} 0 \leq (n^{1/n} - 1) \leq \frac{2}{n^{1/2}}\text{.} \end{equation*}

Now use the squeeze theorem.

Checkpoint 8.3.6.

Let \(x_{1}=1\) and \(x_{n+1}=\sqrt{1+x_{n}}\) for \(n \geq 1\text{.}\) Show that \((x_n)\) converges and compute its limit.

Solution.

We claim that \(x_n\) converges. First, we show that it is increasing by induction, that is, that \(x_{n+1}\geq x_{n}\) for all \(n\geq 1\text{.}\) For the base case, we have \(x_{1}=1\leq \sqrt{1+1}=x_{2}\text{.}\) Now suppose we have shown \(x_{n+1}\geq x_{n}\) for some integer \(n\geq 1\text{.}\) Then

\begin{align*} x_{n+2} -x_{n+1} \amp =\sqrt{1+x_{n+1}}-\sqrt{1+x_{n}}\\ \amp = \left(\sqrt{1+x_{n+1}}-\sqrt{1+x_{n}}\right)\cdot\frac{\sqrt{1+x_{n+1}}+\sqrt{1+x_{n}}}{\sqrt{1+x_{n+1}}+\sqrt{1+x_{n}}}\\ \amp = \frac{1+x_{n+1}-(1+x_{n})}{\sqrt{1+x_{n+1}}+\sqrt{1+x_{n}}} = \frac{x_{n+1}-x_{n}}{\sqrt{1+x_{n+1}}+\sqrt{1+x_{n}}} \end{align*}

Since the denominator is always positive, we just need to check that the numerator is positive, but \(x_{n+1}-x_{n}\geq 0\) by the inductive hypothesis, and so we have \(x_{n+2} -x_{n+1} \geq 0\text{.}\) This proves the inductive step and hence the claim.

Now we need to show \(x_n\) is bounded above. Note that if \(x_n\) did converge to some number \(L\text{,}\) then

\begin{equation} L=\lim_{n\rightarrow\infty}x_{n} =\lim_{n\rightarrow\infty}x_{n+1} =\lim_{n\rightarrow\infty}\sqrt{1+x_{n}} =\sqrt{1+L}\label{e_L_sqrt_1_L_}\tag{8.3.1} \end{equation}

and squaring both sides and moving everything to the left gives \(L^2-L-1=0\text{,}\) so the only possible values for \(L\) are \(\frac{1\pm\sqrt{5}}{2}\text{.}\) Since \(L=\sqrt{1+L}> 0\text{,}\) we must have \(L=\frac{1+\sqrt{5}}{2}\text{.}\) Thus, since \(x_n\) is increasing, it is natural to guess that \(x_n\) is bounded above by \(L\text{.}\) Let's prove this by induction. For \(n=1\text{,}\) we have \(1\leq L\) already, so the base case is true. For the induction step, assume \(x_{n}\leq L\) for some \(n\text{.}\) Then

\begin{equation*} x_{n+1}=\sqrt{1+x_{n}}\leq \sqrt{1+L}\stackrel{(7.3.1)}{=}L\text{.} \end{equation*}

This proves the induction step, and so \((x_n)\) is bounded above and is increasing, thus \((x_n)\) converges and its limit is \(\frac{1+\sqrt{5}}{2}\text{.}\)

Checkpoint 8.3.7.

Let \(x_1=\frac{1}{4}\) and \(x_{n+1} = \sqrt{x_{n}(1-x_{n})}\) for \(n \geq 1\text{.}\) Show that \((x_n)\) converges and compute its limit. (Hint: First show that \(0\lt x_{n}\leq \frac{1}{2}\) for all \(n\text{.}\) The AM–GM inequality may come in useful.)

Solution.

We first show by induction that \(0 \lt x_n \leq 1/2\) for all \(n\text{.}\) The case \(n=1\) is trivial. Suppose inductively that for some \(n\) we have \(0 \lt x_n \leq 1/2\text{.}\) Then \(x_{n+1} =\sqrt{x_{n}(1-x_{n})}>0\) and, by the AM-GM inequality,

\begin{equation*} x_{n+1}=\sqrt{x_{n}(1-x_{n})}\leq \frac{x_{n}+1-x_{n}}{2} = \frac{1}{2}\text{.} \end{equation*}

thus \(x_{n}\leq \frac{1}{2}\) too, establishing the inductive step.

Now we claim that \((x_{n})\) is increasing:

\begin{align*} x_{n+1}-x_{n} \amp =\sqrt{x_{n}(1-x_{n})}-x_{n} = \left(\sqrt{x_{n}(1-x_{n})}-x_{n}\right)\frac{\sqrt{x_{n}(1-x_{n})}+x_{n}}{\sqrt{x_{n}(1-x_{n})}+x_{n}}\\ \amp = \frac{x_{n}(1-x_{n})-x_{n}^{2}}{\sqrt{x_{n}(1-x_{n})}+x_{n}} = \frac{x_{n}(1-2x_{n})}{\sqrt{x_{n}(1-x_{n})}+x_{n}} \geq 0 \end{align*}

since we've already established that \(0 \lt x_n \leq 1/2\) for all \(n\text{.}\) Thus, \((x_{n})\) is increasing and bounded, and thus convergent. Let \(L\) be its limit. Then

\begin{equation*} L=\lim_{n\rightarrow \infty}x_{n} =\lim_{n\rightarrow \infty}x_{n+1} =\lim_{n\rightarrow \infty}\sqrt{x_{n}(1-x_{n})} =\sqrt{L(1-L)} \end{equation*}

squaring both sides and moving everything to the left gives

\begin{equation*} 2L^2-L=0\text{.} \end{equation*}

The only solutions to this are \(L=0\) and \(L=\frac{1}{2}\text{.}\) Since \((x_{n})\) is increasing, we have \(x_{n}\geq x_{1} = \frac{1}{4}>0\text{,}\) so \(L\geq \frac{1}{4}\text{,}\) so \(x_{n}\) can't converge to zero, thus \(L=\frac{1}{2}\text{.}\)

Checkpoint 8.3.8.

Let \(x_{1}=\sqrt{2}\) and \(x_{n+1}=\sqrt{2+\sqrt{x_{n}}}\) for \(n \geq 1\text{.}\) Show that \((x_n)\) converges and find the quartic equation of which the limit is a root. (When showing it is bounded above, you can try making an educated guess as to what an upper bound might be, try some values out!)

Solution.

First we claim that \(x_n >0\) for all \(n\) (so that \(\sqrt{x_n}\) makes sense). This is a simple exercise using induction. Next we claim that \((x_{n})\) is increasing. We prove this also by induction. For \(n=1\text{,}\) we clearly have \(x_{1}=\sqrt{2}\lt \sqrt{2+\sqrt{2}}=x_{2}\text{.}\) This proves the base case. Now suppose that \(x_{n+1}\geq x_{n}\) for some integer \(n\geq 1\text{.}\) Then

\begin{align*} x_{n+2}-x_{n+1} \amp =\sqrt{2+\sqrt{x_{n+1}}}-\sqrt{2+\sqrt{x_{n}}}\\ \amp =\left(\sqrt{2+\sqrt{x_{n+1}}}-\sqrt{2+\sqrt{x_{n}}}\right)\cdot \frac{\sqrt{2+\sqrt{x_{n+1}}}+\sqrt{2+\sqrt{x_{n}}}}{\sqrt{2+\sqrt{x_{n+1}}}+\sqrt{2+\sqrt{x_{n}}}}\\ \amp = \frac{2+\sqrt{x_{n+1}}-2+\sqrt{x_{n}}}{\sqrt{2+\sqrt{x_{n+1}}}+\sqrt{2+\sqrt{x_{n}}}}\\ \amp = \frac{\sqrt{x_{n+1}}-\sqrt{x_{n}}}{\sqrt{2+\sqrt{x_{n+1}}}+\sqrt{2+\sqrt{x_{n}}}}\\ \amp \geq \frac{0}{\sqrt{2+\sqrt{x_{n+1}}}+\sqrt{2+\sqrt{x_{n}}}}=0\text{,} \end{align*}

where in the last line we used the inductive hypothesis that \(x_{n+1}\geq x_{n}\text{,}\) and so \(\sqrt{x_{n+1}}-\sqrt{x_{n}}\geq 0\text{.}\) Thus, \((x_{n})\) is increasing.

Next, we need to show that it is bounded above. We claim that \(x_{n}\leq 2\) for all \(n\text{.}\) We prove this by induction as well. For the base case, \(x_{1}=\sqrt{2}\leq 2\text{.}\) For the inductive step, suppose \(x_{n}\leq 2\) for some \(n\text{.}\) Then

\begin{equation*} x_{n+1}=\sqrt{2+\sqrt{x_{n}}}\leq \sqrt{2+\sqrt{2}}\leq \sqrt{2+2}=\sqrt{4}=2\text{.} \end{equation*}

This establishes the inductive step. Thus, \(x_{n}\) is increasing and bounded above, so it converges. Let \(L\) be the limit. Then, by the rules for limits,

\begin{equation*} L = \sqrt{2+\sqrt{L}}\ \end{equation*}

and therefore \(L\) is a root of the equation \(x^4 - 4x^2 -x + 4 = 0\text{.}\)
