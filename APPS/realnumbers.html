Section 3.1 The Reals

In this course, our aim is to derive as many useful tools and theorems as we can from the fewest assumptions. We could spend a whole year (or years!) on the foundations of mathematics, and try to make sense of, for example, what the number 2 is, but instead we will take for granted the existence the real numbers \(\mathbb{R}\) that you are all familiar with from school. We assume that, if an expression contains a term \(x\) and \(x=y\text{,}\) we can replace \(x\) with \(y\) in that expression. For example, \(3+(5-1)=3+4\) since \(5-1=4\text{.}\)

We will also assume we have the familiar addition operation “\(+\)” and multiplication operation “\(\cdot\)” that combines two numbers \(a\) and \(b\) to make another real number \(a+b\) and \(a\cdot b=ab\) respectively, and which make \(\mathbb{R}\) into what we call a “field”:

Definition 3.1.1.

A field is a set \(F\) along with operations \(+\) and \(\cdot\) so that the following hold:

  • \textbf{Rules of Addition \(+\text{:}\)}.

    For \(a,b,c\in F\text{,}\)

    • (A0).

      \(\displaystyle a+b\in F\)

    • (A1).

      Commutativity: \(a+b=b+a\text{.}\)

    • (A2).

      Associativity: \(a+(b+c)=(a+b)+c\text{.}\)

    • (A3).

      There is an element \(0\in F\) so that \(0+a=a\) for all \(a\in F\text{.}\)

    • (A4).

      There is an element we denote \(-a\in F\) so that \(a+(-a)=0\text{.}\) We write \(b+(-a)=b-a\) for short.

  • \textbf{Rules of Multiplication \(\cdot\text{:}\)}.

    For \(a,b,c\in F\text{,}\)

    • (M0).

      \(\displaystyle a\cdot b\in F\)

    • (M1).

      Commutativity: \(a\cdot b=b\cdot a\)

    • (M2).

      Associativity: \(a\cdot (b\cdot c)=(a\cdot b)\cdot c\)

    • (M3).

      There is an element \(1\in F\) so that \(1\cdot a=a\)

    • (M4).

      If \(a\neq 0\text{,}\) there is a number we denote \(\frac{1}{a}\) so that \(a\cdot\frac{1}{a}=1\text{.}\) We write \(\frac{1}{a}\cdot b=\frac{b}{a}\) for short.

    • (M5).

      Distribution: \(a\cdot (b+c)=a\cdot b+a\cdot c\text{.}\)

We will frequently write \(ab\) for \(a\cdot b\) when there is no chance of ambiguity, so for example, “\(a\) times \(b\)” can be write \(ab\) or \(a\cdot b\text{,}\) but “\(2\) times \(3\)” must be written \(2\cdot 3\) to avoid confusion with the number \(23\text{.}\)

We will take as an axiom that the real numbers \(\mathbb{R}\) form a field with the usual addition and multiplication operations, that is, we assume they hold without proof. There are many other fields apart from the reals, and we will see a few more in this class (you will learn about more interesting ones in future algebra classes).

Without these rules it would be hard to solve any equations. For example, if we want to solve \(3x+2=2x+5\text{,}\) what rules do we need? Below we prove some propositions using only the axioms above that list the “moves” we would usually make in solving such an equation.

Proposition 3.1.2.

The properties (A0),...,(A4) above imply that, for \(x,y,z\in \mathbb{R}\text{,}\)

  1. \(x+y=x+z\) if and only if \(y=z\)

  2. If \(x+y=x\text{,}\) then \(y=0\)

  3. If \(x+y=0\text{,}\) then \(x=-y\)

  4. \(-(-x)=x\text{.}\)

Proof.

  1. If \(y=z\text{,}\) then we can just substitute \(y\) with \(z\) in \(x+y\) to get that this equals \(x+z\text{.}\) Conversely, if \(x+y=x+z\text{,}\) then

    \begin{align*} y \amp \stackrel{(A3)}{=}y+0\stackrel{(A4)}{=}y+(x-x)\stackrel{(A2)}{=}(y+x)-x\stackrel{(A1)}{=}(x+y)-x =(x+z)-x\stackrel{(A1)}{=}(z+x)-x\\ \amp \stackrel{(A2)}{=}z+(x-x) \stackrel{(A4)}{=}z+0 \stackrel{(A3)}{=}z\text{.} \end{align*}

  2. If \(x+y=x\text{,}\) then by (A3), \(x+y=x=x+0\text{,}\) so (a) (with \(z=0\)) implies \(y=0\text{.}\)

  3. If \(x+y=0\text{,}\) then

    \begin{equation*} x+y=0 \stackrel{(A4)}{=}y+(-y) \end{equation*}

    and so (a) (with \(z=-y\)) implies \(x=-y\text{.}\)

  4. By definition,

    \begin{equation*} -x+-(-x)\stackrel{(A4)}{=} 0\stackrel{(A2)}{=}x-x \stackrel{(A1)}{=}-x+x \end{equation*}

    so (a) implies \(-(-x)=x\text{.}\)

Try this proposition on your own (the proof is very similar to the above):

Proposition 3.1.3.

The properties (M0),...,(M5) imply that, for \(x,y,z\in \mathbb{R}\text{,}\) if \(x\neq 0\text{,}\)

  1. \(xy=xz\) if and only if \(y=z\)

  2. If \(xy=x\text{,}\) then \(y=1\)

  3. If \(xy=1\text{,}\) then \(y=\frac{1}{x}\)

  4. \(1/(1/x)=x\text{.}\)

We mention a few more useful facts about the reals. Again, they look obvious, but it takes some work to show that they are derived from the property that \(\mathbb{R}\) being a field. We leave their proofs as an exercise, but remember to use the above axioms and propositions that we have proved so far:

Proposition 3.1.4.

For \(x,y\in \mathbb{R}\text{,}\)

  1. \(0\cdot x=0\text{.}\)

  2. If \(x\neq 0\neq y\text{,}\) then \(xy\neq 0\text{.}\)

  3. \((-x)y=-(xy)=x(-y)\text{.}\)

  4. \((-x)(-y)=xy\text{.}\)
