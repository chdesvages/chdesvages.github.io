Section 15.3 Counting subsets

We will now use the method of overcounting to count the number of subsets of a set \(S\) with \(|S|=n\) which have a given size \(k\text{.}\) Because this number appears over and over again, we give it a special notation:

Definition 15.3.1.

Let \(S\) be set with \(n\) distinct members, and let \(k\) be an integer such that \(0\leq k\leq n\text{.}\) We let

\begin{equation*} {n \choose k} \end{equation*}

denote the number of subsets of \(S\) which have exactly \(k\) members (and the above is read “\(n\) choose \(k\)” because it gives the number of ways we can choose \(k\) members from an \(n\)-element set). (If \(k>n\text{,}\) we set \({n \choose k}=0\)).

Theorem 15.3.2.

For integers \(0\leq k\leq n\text{,}\) we have

\begin{equation} {n \choose k}=\frac{n!}{k!(n-k)!}\text{.}\label{e_choose}\tag{15.3.1} \end{equation}

Liebeck gives a different proof in the book of this theorem, so please look at both. Recall that by convention we declare \(0! = 1\text{.}\)

Notice also that

\begin{equation} {n \choose k}=\frac{n!}{k!(n-k)!} = \frac{n(n-1) \dots(n-k +1)}{k(k-1) \dots 2. 1}\text{.}\tag{15.3.2} \end{equation}

For example \({3 \choose 2} = \frac{3!}{2!1!} = 3\text{,}\) \({6 \choose 3} = \frac{6!}{3!3!} = \frac{6 \cdot 5 \cdot 4}{3 \cdot 2 \cdot 1} =20\) and \({n \choose 0} =1\text{.}\)

Proof.

If \(k=0\text{,}\) there is only one subset of size \(0\text{,}\) which is the empty set, and in this case we see that \(\frac{n!}{0!(n-0)!}=\frac{n!}{n!}=1\text{.}\) Now let \(1\leq k\leq n\) and \(S\) be a set of \(n\) elements. Consider the process where we pick an ordered list \((a_{1}, \dots ,a_{k})\) of \(k\) distinct elements from \(S\text{,}\) one at a time, to form a set \(A=\{a_{1}, \dots ,a_{k}\}\) of \(k\) elements. By Proposition 15.1.6, there are \(\frac{n!}{(n-k)!}\) ways of picking the list \((a_{1}, \dots ,a_{k})\text{.}\) However, this is not the same thing as counting the number of subsets of size \(k\text{,}\) since if we picked \(a_1, \dots ,a_k\) in a different order, they would still give rise to the same set \(A=\{a_{1}, \dots ,a_{k}\}\text{.}\) What we have really counted is the number of ways of picking an ordered list of \(k\) elements, not a set of \(k\) elements. The number of orderings of \(\{a_{1}, \dots ,a_{k}\}\) is \(k!\) by the Ordering Theorem. Thus, we have overcounted the number of subsets of size \(k\) by a factor of \(k!\text{,}\) hence if we divide\(\frac{n!}{(n-k)!}\) by \(k!\text{,}\) we get that the number of subsets of size \(k\) is the right side of (15.3.1).

Note that it's perhaps not completely obvious that the right-hand side of (15.3.1) actually defines a whole number; however once we can identify it as the number of ways of doing something, it has to be a nonnegative integer!

This argument leads us to a new kind of proof, called a combinatorial proof, which is a way of proving a formula by showing that both sides of an equation count the same number of things, rather than trying to prove establish the equation directly via algebra. Let's start with an easy example:

Example 15.3.3.

For \(0 \leq k \leq n\) we have

\begin{equation*} {n\choose k} = {n \choose n-k}\text{.} \end{equation*}

This can be easily shown using the formula (15.3.1), but we will prove it instead using one-to-one correspondence (that is, \(m\)-to-one correspondence with \(m=1\)):

Note that if \(S\) is a set of size \(n\) and \(A\) is a subset of size \(k\text{,}\) then \(S\setminus A\) is a set of size \(n-k\text{.}\) Conversely, if \(B\) is a set of size \(n-k\text{,}\) then \(S \setminus B\) is a set of size \(k\text{.}\) Therefore to every set of size \(k\) we can associate a unique set of size \(n-k\) and vice-versa. Thus the sets of size \(k\) and sets of size \(n-k\) are in one-to-one correspondence, so the number of these sets is the same, that is, \({n\choose k} = {n \choose n-k}\text{.}\) (If you need a bit more convincing, check that if \(B = S \setminus A\text{,}\) then \(S \setminus B = A\text{.}\)) Slightly more formally we describe the correspondence \(A \leftrightarrow S \setminus A\) as a bijection between the sets of size \(k\) and the sets of size \(n-k\text{.}\)

Let's prove another identity where it may be less clear how to prove it algebraically.

Corollary 15.3.4.

For \(n\geq 0\text{,}\) we have

\begin{equation*} 2^{n} = \sum_{k=0}^{n} {n\choose k}\text{.} \end{equation*}
Proof.

Let \(S\) be a set of size \(n\text{,}\) let \(A\) be the set of subsets of \(S\text{,}\) and for \(k=0, \dots ,n\text{,}\) let \(A_k\) denote the subsets of \(S\) of size \(k\text{,}\) so \(A_{k}\) has size \({n\choose k}\) by Theorem 15.3.2. These sets are disjoint and their union is all of \(A\text{,}\) so they form a partition of \(A\text{.}\) Thus, the size of \(A\) is the sum of the sizes of the \(A_{k}\text{.}\) Hence, with \(|A|\) denoting the size of \(A\text{,}\)

\begin{equation*} |A|=|A_{1}|+\cdots + |A_{n}|= \sum_{k=0}^{n} |A_{k}|=\sum_{k=0}^{n}{n\choose k}\text{.} \end{equation*}

We also know by Proposition 15.1.2 that \(|A|=2^{n}\text{,}\) and this proves the corollary.

We have surreptitiously introduced the notion of a partition above. More formally, a partition of a nonempty finite set \(A\) is a collection of subsets \(A_1, \dots , A_m\) of \(A\) such that \(\bigcup_{i=1}^m A_i = A\) and such that for \(i \neq j\text{,}\) \(A_i \cap A_j = \emptyset\text{.}\) In other words, every \(x \in A\) belongs to exactly one \(A_i\text{.}\)

Suppose we are given \(n\) identical red objects laid out along a line, and we are told to intersperse \(m\) identical blue objects amongst them. (We are allowed to begin and end our new chain of objects with blue objects if we wish.) How many ways of doing this are there?

Proposition 15.3.5.

– Linear interspersion principle. The number of ways of interspersing \(m\) identical blue objects amongst \(n\) identical red objects laid along a line is \(n+m \choose m\text{.}\)

Proof.

Each linear arrangement combining the \(m\) blue objects and the \(n\) red objects corresponds to choosing a subset of size \(m\) (the blue objects) from a set of size \(m+n\) (all the objects). There are \(n+m \choose m\) such choices, and hence \(n+m \choose m\) ways of interspersing \(m\) identical blue objects amongst \(n\) identical red objects laid along a line.

Example 15.3.6.

How many ways are there to express the number \(9\) as a sum of three nonnegative integers?

This uses a clever trick which reduces matters to the linear interspersion principle. (It might look intimidating to think it up on your own, but now you'll know it!).

Consider a sum \(x+y+z=9\) with \(0\leq x,y,z\leq 9\) and replace \(x,y,z\) with the number of \(1\)'s that add up to \(x\text{,}\) \(y\) and \(z\) respectively separated by \(+\)'s. For example, \(4+3+2=9\) and \(5+0+4\) correspond to

\begin{equation*} 1\;\; 1\;\; 1\;\; 1\;\; + \;\;1\;\; 1\;\; 1\;\; +\;\;1\;\; 1\;\; \;\;\;\;\;\mbox{ and } \;\;\;\;\; 1\;\; 1\;\; 1\;\; 1\;\; 1 \;\; + \;\; + \;\; 1\;\; 1\;\; 1\;\; 1\;\;\text{.} \end{equation*}

Thus, the number of solutions to \(x+y+z=9\) is exactly the number of ways to intersperse 2 +'s amongst 9 1's laid out in a row. This is the case \(n=9\text{,}\) \(m=2\) of the linear interspersion principle, and is the number of such solutions is thus \({9+2 \choose 2} = {11\choose 2}\text{.}\)

Example 15.3.7.

How many ways are there to place \(9\) identical red balls in three boxes?

This is the previous example in disguise! Put the \(9\) balls in a line, and introduce two “dividers” to delineate the “gaps” between the three boxes. So, for example

\begin{equation*} B\;B\;B\;B\; |\; B\; |\; B\; B\;B\;B \end{equation*}

denotes placing \(4\) balls in box \(1\text{,}\) \(1\) ball in box \(2\) and \(4\) balls in box 3. The number of ways of inserting two dividers in a line of \(9\) balls is therefore \({11\choose 2}\text{.}\)

Now try your hand at combinatorial proofs in the following exercises. The second one will be a workshop problem.

Checkpoint 15.3.8.

Given \(S=\{1,2, \dots ,n\}\) and \(0\leq j\lt n\text{,}\) show that number of subsets of \(S\) of the form \(\{a,a+1, \dots ,a+j\}\subseteq S\) is \({n\choose 2} + n\text{.}\)

Solution.

Note that a set of the form \(\{a,a+1, \dots ,a+j\}\subseteq S\) is uniquely determined by the first and last element of the set, which could be any two numbers, or the same number if \(j=0\text{.}\) Thus, the total number of sets of this form is the same as counting the number of subsets of size \(2\) plus the number of sets of size \(1\text{,}\) and adding them together, hence the total number is

\begin{equation*} {n\choose 2} + n\text{.} \end{equation*}
Checkpoint 15.3.9.

Let \(1 \leq r \leq n\text{.}\) Prove that

\begin{equation} \begin{pmatrix} n+1 \\ r \end{pmatrix}= \begin{pmatrix}n \\ r \end{pmatrix} + \begin{pmatrix} n \\ r-1\end{pmatrix}\text{.}\label{e_n_1_r}\tag{15.3.3} \end{equation}

(You may be familiar with this result if you know about Pascal's triangle.) First prove this by induction using (15.3.1). But then can you give another proof by counting a single quantity in two different ways?

Use this result to show that for \(2 \leq r \leq n\text{,}\)

\begin{equation*} \begin{pmatrix}n+2 \\ r \end{pmatrix} = \begin{pmatrix}n \\ r \end{pmatrix} + 2\begin{pmatrix}n \\ r-1 \end{pmatrix} + \begin{pmatrix}n \\ r-2 \end{pmatrix}\text{.} \end{equation*}

How can you further generalise these results?
