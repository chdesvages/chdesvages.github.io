Section 2.7 Exercises

The relevant exercises in Liebeck's book are in Chapter 19.

Checkpoint 2.7.1.

Consider the following functions. Decide whether each composition is 1-1, onto, both or neither. Do any of the compositions have an inverse function?

  1. \(f_1 \colon \R \to \Z\) defined by \(f_1(x) = \lfloor x \rfloor\)

  2. \(f_2 \colon \Z \to \Z\) defined by \(f_2(x) = x+1\)

  3. \(f_3 \colon \Z \times \Z \to \Z\) defined by \(f_3(x,y) = xy\)

  4. \(f_4 \colon \R \to \Z \times \Z\) defined by \(f_4(x) = (\lfloor x \rfloor, \lceil x \rceil)\)

  5. \(f_5 \colon \{ 1,\dots,10\} \to \{0,1\}\) defined by \(f_5(x) = \begin{cases}0 \amp \text{ if } x \text{ is even } \\1 \amp \text{ if } x \text{ is odd } \end{cases}\)

  6. \(f_6 \colon \{0,1\} \to \{ 1,\dots,10\}\) defined by \(f_6(0) = 10\text{,}\) \(f_6(1) = 7\)

Solution.

We can form the composition of \(f \colon A \to B\) and \(g \colon C \to D\) if and only if \(B=C\text{.}\) In some of the above examples we have that \(B \subset C\text{.}\) In this case, although we can morally do the composition, in practice we would need to do the composition \(g \circ f' \circ f \colon A \to D\) where \(f' \colon B \to C\) is defined by \(f'(x)=x\text{.}\)

Therefore the valid compositions are

  1. \(f_2 \circ f_1 \colon \R \to \Z\) defined by \(f_2 \circ f_1(x) = \lfloor x \rfloor + 1\text{.}\) This is not 1-1 since 1 and 1.5 have the same image. It is onto, because \(n-1\) maps to \(n\) for every \(n \in \Z\text{.}\)

  2. \(f_2 \circ f_2 \colon \Z \to\Z\) defined by \(f_2 \circ f_2(x) = x+2\text{.}\) It is both 1-1 and onto, so is a bijection. Its inverse is the function \(f'\colon \Z \to \Z\) defined by \(f'(x) = x-2\text{.}\)

  3. \(f_2 \circ f_3 \colon \Z \times \Z \to \Z\) defined by \(f_2 \circ f_3(x,y) = xy+1\text{.}\) Not 1-1, since \((1,1)\) and \((-1,-1)\) have the same image. It is onto, since \((x-1,1)\) maps to \(x\) for all \(x \in \Z\text{.}\)

  4. \(f_3 \circ f_4 \colon \R \to \Z\) defined by \(f_3 \circ f_4 (x) = \lfloor x \rfloor \lceil x \rceil\text{.}\) Not 1-1, since 0.1 and 0.2 have the same image. Not onto, because we can only get products of consecutive integers. So, for example, there is no \(x\) which maps to \(14 = 2 \times 7\text{.}\)

  5. \(f_6 \circ f_5 \colon \{1,\dots, 10\} \to \{1,\dots, 10\}\) defined by \(f_6 \circ f_5(x) = \begin{cases}10 \amp \text{ if } x \text{ is even } \\7 \amp \text{ if } x \text{ is odd } \end{cases}\text{.}\) This is not 1-1 since 2 and 4 have the same image. It is not onto because nothing maps to 1.

  6. \(f_5 \circ f_6 \colon \{0,1\} \to \{0,1\}\) defined by \(f_5 \circ f_6(0) = 0\text{,}\) \(f_5 \circ f_6(1) = 1\text{.}\) This is both 1-1 and onto and is its own inverse function. (So just because \(f \circ g\) is bijective, doesn't mean \(g \circ f\) is bijective!)

Checkpoint 2.7.2.

Let \(f:X\rightarrow Y\) be a function.

  1. Show that \(f\) is injective if and only if for all sets \(Z\) and functions \(g,h:Z\rightarrow Y\text{,}\) \(f\circ g=f\circ h\) implies \(g=h\text{.}\)

  2. Show that \(f\) is surjective if and only if for all sets \(Z\) and functions \(g,h:Y\rightarrow Z\text{,}\) \(g\circ f = h\circ f\) if and only if \(g=h\text{.}\)

Checkpoint 2.7.3.

Let \(f:X\rightarrow Y\text{.}\)

  1. Show that \(f(f^{-1}(B))\subseteq B\) for all \(B\subseteq Y\text{,}\) and give an example of an \(f:X\rightarrow Y\) and a set \(B\subseteq Y\) where \(f(f^{-1}(B))\neq B\text{.}\) Solution.

    Note that \(f^{-1}(B)=\{x\in X|f(x)\in B\}\text{,}\) so if \(x\in f^{-1}(B)\text{,}\) \(f(x)\in B\text{,}\) hence \(f(f^{-1}(B))\subseteq B\text{.}\)

    If we let \(f=\cos:\mathbb{R}\rightarrow \mathbb{R}\) and \(B=[2,3]\text{,}\) then \(f^{-1}(B)=\emptyset\text{,}\) so \(f(f^{-1}(B))=\emptyset\neq B\text{.}\)

  2. Show that \(f(f^{-1}(B))=B\) for all \(B\subseteq Y\) if and only if \(f\) is surjective. Solution.

    Suppose \(f\) is surjective. Then for all \(y\in B\) there is \(x\in X\) so that \(f(x)=y\text{.}\) This means \(x\in f^{-1}(B)\) since \(f(x)=y\in B\text{.}\) This means \(y=f(x)\in f(f^{-1}(B))\) for all \(y\in B\text{,}\) thus \(B\subseteq f(f^{-1}(B))\text{,}\) and by (a) we then know \(B = f(f^{-1}(B))\text{.}\)

Checkpoint 2.7.4.

Let \(f:X\rightarrow Y\text{.}\)

  1. Show that \(f^{-1}(f(A))\supseteq A\) for all \(A\subseteq X\text{,}\) and show that these two sets can be unequal. Solution.

    Note that \(f^{-1}(f(A))\) is the set of \(z\in X\) for which \(f(z)\in f(A)\text{,}\) which includes \(A\text{.}\) If we let \(f=\cos:\mathbb{R}\rightarrow \mathbb{R}\) again but let \(A=[0,2\pi]\text{,}\) then \(f(A)=[-1,1]\text{,}\) and \(f^{-1}(f(A))=\mathbb{R}\neq A\text{.}\)

  2. Show that \(f^{-1}(f(A))=A\) for all \(A\subseteq X\) if and only if \(f\) is injective.

Checkpoint 2.7.5.

Let \(X\) and \(Y\) be finite sets, \(m\in \mathbb{N}\text{,}\) and suppose \(f:X\rightarrow Y\) is a function such that for all \(y\in Y\text{,}\) \(|f^{-1}(\{y\})|=m\) . Then

\begin{equation*} |X|=m\cdot |Y|\text{.} \end{equation*}
Solution.

If we number the elements of \(Y\) as \(y_{1},...,y_{n}\) where \(n=|Y|\text{.}\) Then the sets \(f^{-1}(\{y_{1}\}),f^{-1}(\{y_{2}\}),...,f^{-1}(\{y_{n}\})\) partition \(X\text{.}\) Thus,

\begin{equation*} |X|=\sum_{i=1}^{n}| f^{-1}(\{y_{i}\})| = \sum_{i=1}^{n}m=nm=m|Y| \end{equation*}
Checkpoint 2.7.6.

Let \(f:X\rightarrow Y\) be a function and \(A,B\subseteq X\text{.}\) Show that \(f(A\cup B)=f(A)\cup f(B)\text{.}\) Give an example of when we can have \(f(A\cap B)\neq f(A)\cap f(B)\text{.}\)

Checkpoint 2.7.7.

Let \(f:X\rightarrow Y\) be a function. For \(a,b\in X\text{,}\) declare \(a\sim b\) if \(f(a)=f(b)\text{.}\) Show that \(\sim\) defines an equivalence relation on \(X\text{.}\)

Solution.

Claim:Let \(f:X\rightarrow Y\) be a function. For \(a,b\in X\text{,}\) delcare \(a\sim b\) if \(f(a)=f(b)\text{.}\) Then \(\sim\) defines an equivalence relation on \(X\text{.}\)

Proof.

We need to verify reflexivity, symmetry, and transitivity:

  • (Reflexivity) Let \(x\in X\text{.}\) Since \(f(x)=f(x)\text{,}\) we have \(x\sim x\text{,}\) so reflexivity holds.

  • (Symmetry) Let \(x,y\in X\) and suppose \(x\sim y\text{.}\) Then \(f(x)=f(y)\text{,}\) so \(f(y)=f(x)\text{,}\) thus \(y\sim x\) and symmetry holds.

  • (Transitivity) Let \(x,y,z\in X\) and suppose \(x\sim y\) and \(y\sim z\text{.}\) Then \(f(x)=f(y)=f(z)\text{,}\) so \(f(x)=f(z)\text{,}\) hence \(x\sim z\text{.}\) This proves transitivity.

Checkpoint 2.7.8.

Let \(f:X\rightarrow Y\) and \(\sim\) be as in the previous problem. For \(y\in f(X)\text{,}\) define \(F(y)=f^{-1}(\{y\})\) (where recall \(f^{-1}(A)=\{x\in X: f(x)\in A\}\)). Show that \(F\) is a bijection between \(f(X)\) and the set of equivalence classes in \(X\) under the relation \(\sim\text{.}\) Hint: As a first step, you'll need to show the values of \(F\) are indeed equivalence classes.

Solution.

For short, we will write \(f^{-1}(y)\) to mean \(f^{-1}(\{y\})\text{.}\) Claim: Let \(S\) denote the set of equivalence classes under \(\sim\text{.}\) Let \(y\in f(X)\) and define \(F(y)=f^{-1}(y)\text{.}\) Then \(F:f(X)\rightarrow S\) is a bijection.

Proof.

For \(y\in f(X)\text{,}\) \(f^{-1}(y)\) is an equivalence class: since \(y\in f(X)\text{,}\) \(y=f(x)\) for some \(x\in X\text{,}\) and if \([x]\) denotes the equivalence class for \(x\text{,}\)

\begin{equation*} [x]=\{z\in X:x\sim z\} = \{z\in X:f(z)=f(x)\} = \{z\in X:f(z)=y\} =f^{-1}(\{y\})\text{.} \end{equation*}

hence \(F\) is a function from \(f(X)\) to \(S\text{.}\) We need to show that it is injective and surjective:

  • (Injectivity) Let \(y,z\in f(X)\) and suppose \(F(y)=F(z)\text{,}\) we will show that \(y=z\text{.}\) Since \(F(y)=F(z)\text{,}\) this means \(f^{-1}(y)=f^{-1}(z)\text{.}\) Let \(x\in f^{-1}(y)\text{,}\) so \(f(x)=y\text{.}\) But then \(x\in f^{-1}(y)=f^{-1}(z)\text{,}\) so \(f(x)=z\text{,}\) thus \(y=f(x)=z\text{.}\) This proves injectivity.

  • (Surjectivity) Let \([x]\in S\) be an equivalence class. Let \(y=f(x)\text{.}\) Then \([x]=f^{-1}(y)\) and \(y\in f(X)\text{.}\) Thus, \(F(y)=f^{-1}(y)=[x]\text{.}\) This implies surjectivity.

Checkpoint 2.7.9.

Show that \(f:\mathbb{R}\rightarrow \mathbb{R}\) defined by \(f(x)=x^{2n+1}\) is bijective for \(n\in\mathbb{N}\text{.}\)

Solution.

Recall that

\begin{equation*} 1+z+\cdots + z^{n} = \frac{1-z^{n+1}}{1-z} \end{equation*}

and so if \(x\neq y\text{,}\) and \(x\neq 0\text{,}\) applying this formula with \(z=(y/x)^2\text{,}\)

\begin{equation*} x^{2n+1} - y^{2n+1} = x^{2n+1} \left(1-\left(\frac{y}{x}\right)^{2n+1}\right) = x^{2n+1} \left(1-\left(\frac{y}{x}\right)^2\right)\left(1+\left(\frac{y}{x}\right)^2+\cdots + \left(\frac{y}{x})^{2n}\right)\right)\text{.} \end{equation*}

and this can't be zero if \(x\neq 0\) and \(x\neq y\text{.}\) If \(x=0\) and \(y\neq x=0\text{,}\) then we clearly have \(f(x)\neq f(y)\text{.}\) Thus, \(f\) is injective.

Checkpoint 2.7.10.

Quite Challenging! Find bijections between the following sets:

  1. \(\R\) and \((0,\infty)\) Solution.

    Let \(f:(0,\infty) \rightarrow \R\) be \(f(x)=x-\frac{1}{x}\text{.}\) Then \(f\) is increasing, so it is injective. Furthermore, if \(y\in \R\text{,}\) we can solve \(f(x)=y\) by solving the equation

    \begin{equation*} y= x-\frac{1}{x}\text{,} \end{equation*}

    and rearranging, this is equivalent to

    \begin{equation*} 0=x^2-yx-1\text{,} \end{equation*}

    which has solution

    \begin{equation*} x=\frac{y\pm\sqrt{y^2+4}}{2}\text{,} \end{equation*}

    and one of these solutions is in \((0,\infty)\text{,}\) thus \(f\) is surjective, and hence bijective.

  2. \((0,\infty)\) and \((1,\infty)\) Solution.

Checkpoint 2.7.11.

The following argument supposedly shows that the reflexivity condition is unnecessary for a nonempty relation \(\sim\text{,}\) that is, it can be derived from symmetry and transitivity:

(a) Suppose \(a\sim b\text{.}\)

(b) By symmetry, \(b\sim a\text{.}\)

(c) Since \(a\sim b\) and \(b\sim a\text{,}\) by transitivity, \(a\sim a\text{.}\) Therefore, \(\sim\) is reflexive.

What's wrong with this argument?

Solution.

The problem with this argument is (a), as it asserts that, for every \(a\text{,}\) there is a \(b\) so that \(a\sim b\text{.}\) But not every relation requires every element to be related to something.

A counter example could be the relation on \(\{1,2\}\) where only \(1\sim 1\) (so \(2\not\sim 2\) and \(2\not\sim 1\)). Then it is symmetric and transitive, but not reflexive.
