Section 5.5 Exercises

The relevant exercises in Liebeck are in Chapter 8.

Checkpoint 5.5.1.

(Aug 2018 Exam) Show that, for all \(n\geq 0\text{,}\) \(\sum_{k=1}^{n} k\cdot k! = (n+1)!-1\text{.}\)

Solution.

The base case is \(\sum_{k=1}^{1} k\cdot k!=1=2!-1\text{.}\) For the induction step, assuming the \((n-1)\)st case is true,

\begin{equation*} \sum_{k=1}^{n} k\cdot k! =\sum_{k=1}^{n-1} k\cdot k! + n\cdot n! =n!-1+ n\cdot n! =(n+1)\cdot n! -1 = (n+1)!-1\text{.} \end{equation*}
Checkpoint 5.5.2.

Show that for all \(n\geq 0\text{,}\) \(\sum_{k=1}^{n} k(k+1) = \frac{n(n+1)(n+2)}{3}\text{.}\)

Solution.

The base case is \(\sum_{k=1}^{1} k(k+1) = \frac{1(1+1)(1+2)}{3}\text{.}\) For the induction step assume \(\sum_{k=1}^{n} k(k+1) = \frac{n(n+1)(n+2)}{3}\) and consider

\begin{equation*} \sum_{k=1}^{n+1} k(k+1) = \sum_{k=1}^{n} k(k+1) +(n+1)(n+2) \end{equation*}
\begin{equation*} = \frac{n(n+1)(n+2)}{3}+(n+1)(n+2) =\frac{(n+1)(n+2)(n+3)}{3}\text{.} \end{equation*}
Checkpoint 5.5.3.

Show that \(n!\leq n^{n}\) for all \(n\geq 1\)

Solution.

The base holds immediately. Assume \(n\geq 1\) is so that \(n^{n}\geq n!\text{.}\) Then

\begin{equation*} (n+1)!=n! (n+1)\leq n^{n} (n+1)\lt (n+1)^{n}(n+1)=(n+1)^{n+1}\text{.} \end{equation*}
Checkpoint 5.5.4.

Suppose \(x_{1}=1\) and \(x_{n+1} = \sqrt{1+2x_{n}}\) for all \(n\geq 1\text{.}\) Show \(x_{n}\leq 4\) for all \(n\text{.}\)

Solution.

This clearly holds for \(n=1\text{.}\) Assume \(x_{n}\leq 4\text{,}\) then

\begin{equation*} x_{n+1}= \sqrt{1+2\cdot x_{n}}\leq \sqrt{1+2\cdot 4}=\sqrt{9}=3\lt 4 \end{equation*}

and thus the induction step also holds.

Checkpoint 5.5.5.

Let \(f_{1}=f_{2}=1\) and \(f_{n}=f_{n-1}+f_{n-2}\) for \(n>2\text{.}\)

  1. Show that \(3|f_{4n}\) for all \(n\geq 1\text{.}\)

  2. Show that \(1\leq f_{n+1}/f_{n}\leq 2\) for all \(n\geq 1\text{.}\)

Solution.

  1. We prove by induction. We can compute that \(f_{4}=3\text{,}\) which establishes the base case \(n=1\text{.}\) Now assume \(3|f_{4n}\) for some \(n\geq 1\text{,}\) we'll show \(3|f_{4(n+1)}\text{:}\)

    \begin{equation*} f_{4(n+1)}=f_{4n+3}+f_{4n+2} =2f_{4n+2}+f_{4n+1} =2(f_{4n+1}+f_{4n})+f_{4n+1} =3f_{4n+1}+2f_{4n}\text{.} \end{equation*}

    By assumption, \(3|f_{4n}\text{,}\) and clearly \(3|3f_{4n+1}\text{,}\) and so \(3|f_{4(n+1)}\text{,}\) which proves the induction step and hence the claim.

  2. It is clear the inequalities hold for \(n=1\text{.}\) Assume we have shown it to hold for some \(n\text{.}\) Then

    \begin{equation*} \frac{f_{n+2}}{f_{n+1}}=1+\frac{f_{n}}{f_{n}}\leq 1+1=2 \end{equation*}

    and

    \begin{equation*} \frac{f_{n+2}}{f_{n+1}}=1+\frac{f_{n}}{f_{n}}\geq 1+0=1\text{.} \end{equation*}

    This proves the induction step and hence the claim.

Checkpoint 5.5.6.

Let \(F_{n}=2^{2^{n}}+1\) for \(n\geq 0\text{.}\) Prove that for \(n>0\text{,}\)

\begin{equation*} F_{n} = F_{n-1}\cdots F_{0}+2\text{.} \end{equation*}
Solution.

We prove by induction. The base case \(n=1\) is immediate. Suppose the statement is true for some \(n\geq 1\text{.}\) Then

\begin{equation*} F_{n}\cdot F_{n-1}\cdots F_{0}+2 = F_{n}(F_{n}-2)+2 =F_{n}^{2}-2F_{n}+2 =(F_{n}-1)^2+1 =(2^{2^{n}})^2+1 = 2^{2^{n+1}}+1 =F_{n+1}\text{.} \end{equation*}
Checkpoint 5.5.7.

Find a formula for the number of diagonals in a convex polygon with \(n\geq 3\) vertices.

Solution.

Let \(f(n)\) denote the number of diagonals in a convex polygon of \(n\) vertices. We will find a recurrence relation that \(f\) satisfies.

Suppose we have a convex polygon with \(n+1\) vertices. Draw an edge between the second vertex \(a\) and last vertex \(c\text{,}\) so splits the polygon into a triangle between \(a\text{,}\) \(c\text{,}\) and the first vertex \(c\) and a polygon with \(n\) vertices. The number of diagonals in this polygon (which are diagonals for the original polygon) are \(f(n)\) many. The ones that are diagonals for the original polygon that weren't counted are the ones that have \(b\) as an endpoint (of which there are \(n+1-3=n-2\) many) and the edge that we first drew. Thus the total is

\begin{align*} f(n+1) \amp =f(n)+n-2+1=f(n)+n-1 = f(n-1) + n-2+n-1=\cdots\\ \amp = f(3) + \sum_{k=2}^{n-1}k =0+\sum_{k=1}^{n-1}k-1 = \frac{n(n-1)}{2}-1\text{.} \end{align*}
Checkpoint 5.5.8.

Show that for all integers \(n\geq 3\text{,}\) there are distinct integers \(x_{1},...,x_{n}\) so that

\begin{equation*} \sum_{k=1}^{n} \frac{1}{x_{k}}=1\text{.} \end{equation*}

(It is an open question whether you can always do this with \(x_{n}\) distinct odd numbers.)

Solution.

We prove this by induction. For the base case \(n=3\text{,}\) we see that

\begin{equation*} 1=\frac{1}{2}+\frac{1}{3}+\frac{1}{6}\text{.} \end{equation*}

Suppose the claim is true for some integer \(n\geq 3\text{.}\) Let \(x_{1},...,x_{n}\) be so that

\begin{equation*} \sum_{k=1}^{n} \frac{1}{x_{k}}=1\text{.} \end{equation*}

Note that we must have \(x_{k}>1\) for all \(k\text{.}\) Then

\begin{equation*} \sum_{k=1}^{n} \frac{1}{2x_{k}}=\frac{1}{2}\text{.} \end{equation*}

Hence,

\begin{equation*} \frac{1}{2} + \frac{1}{2x_{1}}+\cdots + \frac{1}{2x_{n}}=\frac{1}{2}+\frac{1}{2} =1\text{,} \end{equation*}

Note that as \(x_{k}\geq 2\) for all \(k\text{,}\) \(2x_{k}\neq 2\) for all \(k\text{,}\) thus the integers \(2,2x_{1},...,2x_{n}\) are \(n+1\) distinct integers whose reciprocals add up to one, which proves the induction step and hence the claim.

Checkpoint 5.5.9.

Joseph Bertrand conjectured in 1845 (and it was proven by Chebychev in 1852) that, if \(p_{1},p_{2},..\text{.}\) are the primes in ascending order, then

\begin{equation*} p_{n+1}\lt 2p_{n}\text{.} \end{equation*}

This is called Bertrand's postulate. Using this result, show that for all \(n\geq 4\) we have

\begin{equation*} p_{n} \lt \sum_{k=1}^{n-1} p_{k}\text{.} \end{equation*}
Solution.

We prove this by induction. For \(n=4\text{,}\) we have

\begin{equation*} p_{4}=7 \lt 2+3+5 = \sum_{k=1}^{3} p_{k}\text{.} \end{equation*}

This establishes the base case. Now assume the statement is true for some integer \(n\text{.}\) Then

\begin{equation*} p_{n+1}\lt 2p_{n} = p_{n}+p_{n} \lt p_{n}+\sum_{k=1}^{n-1} p_{k} = \sum_{k=1}^{n} p_{k}\text{.} \end{equation*}

This proves the induction step and hence the claim.

Checkpoint 5.5.10.

Challenge: In this exercise, we will prove the Arithmetic-Geometric Inequality: for \(x_{1},...,x_{n}\geq 0\text{,}\)

\begin{equation*} (x_{1}\cdots x_{n})^{\frac{1}{n}}\leq \frac{x_{1}+\cdots + x_{n}}{n}\text{.} \end{equation*}

We will do this in several steps.

  1. Prove that for all \(n\in \mathbb{N}\text{,}\) if \(x_{1},...,x_{2^n}\) are positive numbers, then

    \begin{equation*} (x_{1}x_{2}\cdots x_{2^n})^{1/2^{n}} \leq \frac{x_{1}+\cdots + x_{2^n}}{2^n}\text{.} \end{equation*}

    Solution.

    This follows by induction on \(n\text{.}\) The \(n=2\) case was shown just below the statement of [cross-reference to target(s) "t_AM-GM" missing or not unique]. For the induction step, assume the \(n\)th case is true for some \(n\geq 2\text{,}\) we can prove the \((n+1)\)st case using the base case: Let \(a=(x_{1}\cdots x_{2^{n}})^{\frac{1}{2^{n}}}\) and \(y=(x_{2^{n}+1}\cdots x_{2^{n+1}})^{\frac{1}{2^{n}}}\text{.}\) Then

    \begin{equation*} (x_{1}....x_{2^{n+1}})^{\frac{1}{2^{n+1}}} =\sqrt{ab}\leq \frac{a+b}{2} \leq \frac{x_{1}+\cdots + x_{2^{n}}+x_{2^{n}+1}+\cdots + x_{2^{n+1}}}{2^{n+1}} \end{equation*}

    where in the last line we used the induction hypothesis.

  2. Use the previous case to prove the AM-GM inequality. Hint: Let \(2^{m}>n\) and define a new sequence \(y_{1},...,y_{2^{m}}\) with \(y_{i}=x_{i}\) for \(i=1,2,...,n\) and \(y_{i}=A\) otherwise where \(A=\frac{x_{1}+\cdots + x_{n}}{n}\text{.}\) Apply the previous case to the product of the \(y_{i}\) and find an appropriate choice of \(A\) that will imply the inequality for the \(x_{i}\text{.}\) Solution.

    With the notation as in the hint, we see that

    \begin{align*} (x_{1}\cdots x_{n}A^{2^{m}-n})^{\frac{1}{2^{m}}} \amp = (y_{1}\cdots y_{2^{m}})^{\frac{1}{2^{m}}}\\ \amp \leq \frac{y_{1}+\cdots + y_{2^{m}}}{2^{m}}\\ \amp =\frac{x_{1}+\cdots + x_{n} + (2^{m}-n)A}{2^{m}}\\ \amp = \frac{nA+(2^{m}-n)A}{2^{m}}\\ \amp = A \end{align*}

    Taking both sides to the power \(\frac{2^{m}}{n}\text{,}\) we get

    \begin{equation*} (x_{1}\cdots x_{n}A^{2^{m}-n})^{\frac{1}{n}} \leq A^{\frac{2^{m}}{n}}\text{.} \end{equation*}

    Now moving the \(A\) from the left side to the right, we get

    \begin{equation*} (x_{1}\cdots x_{n})^{\frac{1}{n}}\leq A=\frac{x_{1}+\cdots + x_{n}}{n}\text{.} \end{equation*}
