Modern set theory (and mathematics in general) is based on a finite set of axioms called Zermelo–Fraenkel set theory (ZFC). In 1940, Kurt Gödel showed that the CH cannot be disproved assuming ZFC, so it would see then that CH has to be true...but then Paul Cohen showed in 1963 that CH cannot be proved assuming ZFC. In other words, CH is independent of the axioms of mathematics: one can assume it's true or false and it won't contradict anything proven using the ZFC axioms!
