Let \(d=\gcd(n!+1,(n+1)!)\text{.}\) By The Linear Combination Lemma,

\begin{equation*} d \text{ divides } (n+1)(n!+1)-(n+1)! = n+1 \end{equation*}

If \(d>1\) and \(k\lt n+1\) is such that \(d|k\text{,}\) then by The Linear Combination Lemma

\begin{equation*} d \text{ divides } n!+1- k\cdot \frac{n!}{k} =1 \end{equation*}

a contradiction, thus \(d\) cannot divide any number less than \(n+1\) bigger than \(1\text{,}\) so \(d\) is either \(1\) or \(n+1\text{.}\)
