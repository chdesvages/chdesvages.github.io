Solution 16.10.6.1.

(a) Clearly \(y\sim y\) since \(y=\iota y\iota^{-1}\text{,}\) so \(\sim\) is reflexive. If \(y\sim z\text{,}\) then \(y=xzx^{-1}\) for some \(x\in S_{n}\text{.}\) Composing both sides on the left with \(x^{-1}\text{,}\) we get \(x^{-1}y=zx^{-1}\text{,}\) and then composing again on the right gives \(x^{-1}yx=z\text{,}\) and so \(z\) is a conjugate of \(y\text{,}\) hence \(z\sim y\) and we have reflexivity. Finally, if \(y\sim z\) and \(z\sim w\text{,}\) then \(y=xzx^{-1}\) and \(z=sws^{-1}\) for some \(x,s\in S_{n}\text{.}\) Then \(y=xzx^{-1} = xswx^{-1}s^{-1} = xsw(xs)^{-1}\text{,}\) so \(y\sim w\text{.}\) This proves transitivity.

(b) Let \(F(y)=xyx^{-1}\text{.}\) Then \(F\) is surjective since, for every \(y\in S_{n}\text{,}\) \(y=F(x^{-1}yx)\text{.}\) It is injective since, if \(xyx^{-1}=xzx^{-1}\text{,}\) then we can compose both sides on the left with \(x^{-1}\) to get \(yx^{-1}=zx^{-1}\text{,}\) and then compose with \(x\) on the right to get \(y=z\text{.}\)

(c) If we plug in \(x(a_{i})\) (this is the function \(x\) evaluated at \(a_{i}\)), we get \(xcx^{-1}(x(a_{i}))=xc(a_{i})=x(a_{i+1})\) if \(i\lt k\) and when \(i=k\text{,}\) \(xcx^{-1}(x(a_{k}))=xc(a_{k})=x(a_{1})\text{.}\) Thus, \(xcx^{-1}=(x(a_{1})...x(a_{k}))\text{.}\)

In particular, \(xcx^{-1}\) is a cycle of the same length.

(d) If \(y,z\) are conjugates, then they have the same cycle-shape. If \(z=xyx^{-1}\text{,}\) and \(y\in S_{n}\) decomposes into disjoint cycles \(y=c_{1}\cdots c_{k}\text{,}\) then

\begin{equation*} xyx^{-1}=xc_{1}x^{-1} xc_{2}x^{-1}\cdots xc_{k}x^{-1}\text{,} \end{equation*}

so \(xyx^{-1}\) is also a composition of cycles of the same lengths, and so it has the same cycle-shape as \(y\text{.}\)

If \(y\) and \(z\) have the same cycle shape, then they are also conjugate. To see this, note that if \(y=c_1...c_{r}\) is a product of disjoint cycles, we can write \(z=d_{1}...d_{r}\) as a product of cycles so that \(d_{i}\) has the same length as \(c_{i}\text{.}\) If \(c_{i}=(a_{i,1}...a_{i,k_i})\) and \(d_{i}=(b_{i,j},...,b_{i,k_{i}})\text{,}\) define \(x\in S_{n}\) so that \(x(a_{i,j})=b_{i,j}\text{.}\) Then \(xc_ix^{-1}=(x(a_{i,1})...x(a_{i,k_i}))=(b_{i,1},...,b_{i,k_{i}})=d_i\text{,}\) and so \(xyx^{-1}=z\text{.}\)
