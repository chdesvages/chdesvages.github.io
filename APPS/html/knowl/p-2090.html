Since \(f(1),..\text{.}\) runs through each integer \(1,2,...,n\) exactly once, the top row is just a rearrangement of \(\{1,2,...,n\}\text{,}\) so if we rearrange the columns so that the top row is in order, then we will get the chart

\begin{equation*} f^{-1}=\left(\begin{matrix}1 \amp 2 \amp 3 \amp ... \amp n\crf^{-1}(1) \amp f^{-1}(2) \amp f^{-1}(3) \amp ... \amp f^{-1}(n)\cr \end{matrix} \right)\text{.} \end{equation*}
