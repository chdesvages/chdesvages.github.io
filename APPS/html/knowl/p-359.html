We can form the composition of \(f \colon A \to B\) and \(g \colon C \to D\) if and only if \(B=C\text{.}\) In some of the above examples we have that \(B \subset C\text{.}\) In this case, although we can morally do the composition, in practice we would need to do the composition \(g \circ f' \circ f \colon A \to D\) where \(f' \colon B \to C\) is defined by \(f'(x)=x\text{.}\)
