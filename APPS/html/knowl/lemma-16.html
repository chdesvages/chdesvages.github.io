Lemma 16.5.3.

The set \(\mathrm{S}_n\) equipped with the composition rule \(\circ\) has the following properties:

  • Closure: for any \(f\) and \(g\) in \(\mathrm{S}_n\text{,}\) one has \(f\circ g\in\mathrm{S}_n\text{,}\)

  • Associativity: for any \(f\text{,}\) \(g\text{,}\) and \(h\) in \(\mathrm{S}_n\text{,}\) one has

    \begin{equation*} f\circ \Big(g\circ h\Big)=\Big(f\circ g\Big)\circ h\text{,} \end{equation*}

  • Identity element: there is unique permutation \(\iota\in\mathrm{S}_n\) such that

    \begin{equation*} f\circ\iota=\iota\circ f=f \end{equation*}

    for any \(f\in\mathrm{S}_n\) (and we call \(\iota\) the identity permutation),

  • Inverse: for any \(f\in\mathrm{S}_n\text{,}\) there is unique \(f^{-1}\in\mathrm{S}_n\)  such that

    \begin{equation*} f\circ f^{-1}=f^{-1}\circ f=\iota\text{.} \end{equation*}
