Suppose there was \(f:S\rightarrow P(S)\) bijective, let \(B=\{x\in S: x\not\in f(x)\}\text{.}\) Since \(f\) is bijective, it is surjective, and so there is \(x\) so that \(f(x)=B\text{.}\)

  1. If \(x\in B\text{,}\) then by definition of \(B\text{,}\) \(x\not\in f(x)=B\text{,}\) a contradiction.

  2. If \(x\not\in B\text{,}\) then by definition of \(B\text{,}\) \(x\in f(x)=B\text{,}\) again a contradiction.

Since these are the only two possibilities, we have a full contradiction. Thus, there cannot be a function \(f:S\rightarrow P(S)\) bijective, thus \(|S|\neq |P(S)|\text{.}\) Since there is a clear injection \(g:S\rightarrow P(S)\) defined by \(g(x)=\{x\}\text{,}\) we have that \(|S|\leq |P(S)|\text{,}\) and so \(|S|\lt |P(S)|\text{.}\)
