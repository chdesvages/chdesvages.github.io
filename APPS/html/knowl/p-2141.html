Let \(f=(1 \; 2 \; 3)(4 \; 5)\) and \(g=(1 \; 2 \; 3 \; 4)\) be permutaitons in \(S_{5}\text{.}\) To find the cycle notation for \(fg\text{,}\) we do as before: start with \(1\) and look at the values we get by repeatedly plugging into \(fg\text{:}\)

\begin{gather*} fg(1) = f(g(1)) = f(2) = 3,\\ fg(3) = f(g(3))=f(4)=5,\\ fg(5) = f(g(5)) = f(5) = 4,\\ fg(4) = f(g(4)) = f(1) = 2,\\ fg(2) = f(g(2)) = f(3) = 1\text{.} \end{gather*}
