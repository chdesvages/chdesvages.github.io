One can check this by plugging in various values \(a_i\text{.}\) If we plug in \(a_3\text{,}\) for example, remember that when plugging a number into a composition \(f_{1}...f_{n}\text{,}\) we plug the number into \(f_n\) first, then the resulting value into \(f_{n-1}\text{,}\) and so on. So in the above product for \(f\text{,}\) we first plug \(a_3\) into \((a_{1}\; a_{2})\text{,}\) which just returns \(a_{3}\text{.}\) Then we plug that into \((a_{1}\; a_{3})\text{,}\) which gives \(a_{1}\text{.}\) Then we plug that into \((a_{1}\; a_{4})\) ad we get \(a_{4}\text{.}\) But now \(a_{4}\) does not appear in any other \(2\)-cycle, and so they all return \(a_{4}\text{.}\) Hence, \(f(a_{3})=a_{4}\text{,}\) and we can show similarly that \(f(a_{i})=a_{i+1}\) for all \(i\) except \(r\) where \(f(a_{r})=a_{1}\text{.}\)
