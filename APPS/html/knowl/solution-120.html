Solution 11.5.1.1.

Recall that the roots of this polynomial are

\begin{equation*} x= \frac{5\pm z}{2} \end{equation*}

where \(z\) are the solutions to

\begin{equation*} z^2 = 5^2-4\cdot (7-i)\cdot 1 = 25-28+4i = -3+4i\text{.} \end{equation*}

If we set \(z=a+ib\text{,}\) this gives

\begin{equation*} a^2-b^2+2abi = -3+4i\text{.} \end{equation*}

So in particular, \(2ai = 4i\) and \(a^2-b^2 = -3\text{.}\) Moreover,

\begin{equation*} |z^2|= |-3 + 4i| = 5 \end{equation*}

Thus,

\begin{equation*} 5=|z^2|=|z|^2 = \sqrt{a^2+b^2}^2 = a^2+b^2 \end{equation*}

Adding this to \(a^2-b^2=-3\) implies \(2a^2 = 2\text{,}\) so \(a^2=1\text{,}\) and so \(a = \pm 1\text{.}\) Similarly, subtracting \(a^2-b^2=-3\) from the above equation gives \(2b^2 = 8\text{,}\) so \(b=\pm 2\text{.}\) Finally, \(2ab = 4\) implies that \(a\) and \(b\) have the same algebraic sign, so we must have \(a+ib\) is \(1+2i\) or \(-1-2i\) as our two solutions for \(z\text{.}\) Hence,

\begin{equation*} x=\frac{5 \pm (1+2i)}{2} \end{equation*}

so \(x\) is either \(3+i\) or \(2-i\text{.}\)

We can substitute these in to check, e.g. (just the first)

\begin{equation*} (3+i)^2-5(3+i)+7-i = (8+6i)-5(3+i)+7-i =0 \end{equation*}
