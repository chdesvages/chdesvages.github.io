Suppose \((x,y,z)\) was an integer solution. Notice that \(8z+5\) is odd, so \(x^2+2y^2\) is odd as well, thus \(x^2\) is odd, and so \(x\) is odd. Thus, \(x=2n+1\) for some integer \(n\text{.}\) Plugging that back into the above, we get

\begin{equation*} 8z+5 =x^2+2y^2 = (2n+1)^2 +2y^2 = 4n^2 +4n+1 + 2y^2 \end{equation*}

and so

\begin{equation*} 8z+4 = 4n^2+4n+2y^2\text{.} \end{equation*}

Dividing both sides by \(2\text{,}\) we get

\begin{equation*} 4z+2=2n^2+2n+y^3\text{.} \end{equation*}

In particular, \(y^3\) must be even, so \(y\) is even, hence \(y=2m\) for some \(m\text{,}\) so we get

\begin{equation*} 4z+2-2n^2-2n=y^3=8m^3 \end{equation*}

and dividing through by \(2\) gives

\begin{equation*} 2z+1-n^2-n = 4m^3\text{.} \end{equation*}

But notice that \(2z+1-n^2-n = 2z-n(n+1)+1\) is odd, because \(n(n+1)\) is even and so is \(2z\text{,}\) but \(4m^3\) is even, and we get a contradiction. Thus, there are no solutions.
