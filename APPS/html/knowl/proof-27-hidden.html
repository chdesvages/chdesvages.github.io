Suppose \((x,y)\) is a solution. First note that this implies \(x^3>y^3\text{,}\) and so \(x>y\text{.}\) Furthermore, we immediately see that \(x=0\) and \(y=0\) don't lead to solutions, so we can assume that \(x\neq 0 \neq y\) as well.

Factoring, we see that

\begin{equation*} 7 = x^3-y^3= (x-y)(x^2+xy+y^2) \end{equation*}

Thus, \(x-y\) divides \(7\text{,}\) and since \(x>y\text{,}\) this means \(x-y>0\text{,}\) so the only possibilities then are \(x-y=1\) or \(x-y=7\text{.}\) We split into two cases:

  1. \(x-y=1\text{.}\) Then we have that \(x=y+1\text{,}\) and so

    \begin{equation*} 7=x^3-y^3= (y+1)^3-y^3 = 3y^2+3y+1\text{,} \end{equation*}

    thus \(6=3y^2+y\text{,}\) so \(2=y^2+y=y(y+1)\text{,}\) so we see that \(y\) divides \(2\) and \(y+1\) divides \(2\text{.}\) The only way this is possible is if either \(y=1\) or \(y=-2\text{.}\) By plugging these values into the original equation, we see that the solutions are in this case \((x,y)=(2,1)\) and \((-1,-2)\text{.}\)

  2. Suppose \(x-y=7\text{.}\) Then

    \begin{equation*} 7=(x-y)(x^2+xy+y^2)=7(x^2+xy+y^2) \end{equation*}

    implies

    \begin{equation*} 1=(x^2+xy+y^2)\text{.} \end{equation*}

    Note that if \(|x|\geq |y|\text{,}\) then \(xy\geq -|x|\cdot |y|\geq -|x|^2=-x^2\text{,}\) and so the above is at least

    \begin{equation*} 1\geq x^2-x^2=y^2>0 \end{equation*}

    since we are assuming \(y\neq 0\text{.}\) Hence, \(y^2=1\text{,}\) so \(y=\pm 1\text{.}\) Again, we can plug these into our original equation \(7=x^3-y^3\) and we find that \(y=-1\) leads to no solution and \(y=1\) leads to \((x,y)=(2,1)\text{,}\) which we already found. Thus, the solutions are just \((x,y)=(2,1)\) and \((-1,-2)\text{.}\)
