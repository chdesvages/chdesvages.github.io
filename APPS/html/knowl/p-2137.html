How do we find the cycle decomposition in practice? The proof above gives us a roadmap: given a permutation \(f\text{,}\) we know there is an integer \(r\) so that if \(a_{i}= f^{i}(1)\text{,}\) then \(a_{r}=1\text{,}\) and we know that one of the cycles in the decomposition for \(f\) is \((1 \; a_{1}\; \cdots \; a_{r-1})\text{.}\) To find another cycle, pick \(k\not\in \{1,a_{1},...,a_{r-1}\}\) and repeat the process: there is \(s\) so that if \(b_{i}=f^{i}(k)\text{,}\) then \(b_{s}=k\text{,}\) and another cycle in the decomposition is \((k,b_{1},...,b_{s-1})\text{.}\) Repeat this process until each integer is in one of the cycles you've found.
