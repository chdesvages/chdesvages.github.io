Let

\begin{equation*} S=\{x \; | \; \mbox{ there are infinitely many \(n\in\mathbb{N}\) so that } x_n\geq x\} \end{equation*}

and set \(L=\LUB (S)\text{.}\) Since \((x_n)\) is bounded, \(\LUB(S)\) exists. By PropositionÂ 9.6.3, \(L\) is a limit point if we can verify the following:

Claim: For all \(\epsilon>0\) and \(N\in\mathbb{N}\) there is \(n>N\) so that \(|x_{n}-L|\lt \epsilon\text{.}\)

We prove by contradiction. Suppose instead that there exists \(\epsilon>0\) and \(N\in\mathbb{N}\) so that \(|x_{n}-L|\geq \epsilon\) for all \(n> N\text{.}\) In other words, there are at most \(N\) many integers \(n\) (hence only finitely many) for which \(|x_{n}-L|\lt \epsilon\text{,}\) or equivalently

\begin{equation} L-\epsilon\lt x_n\lt L+\epsilon\text{.}\tag{9.6.2} \end{equation}

Since \(L+\epsilon>L=\LUB(S)\) and \(\LUB(S)\) is an upper bound for \(S\text{,}\) we can't have \(L+\epsilon\in S\text{,}\) so by the definition of \(S\text{,}\) there are only finitely many \(n\in\mathbb{N}\) so that

\begin{equation} x_n\geq L+\epsilon\tag{9.6.3} \end{equation}

Thus, there are only finitely many \(n\) satisfying (9.6.2) or (9.6.3), so there can only be finitely many \(n\) so that \(x_n>L-\epsilon\text{.}\) But then \(L-\epsilon\geq \LUB(S)\text{,}\) since if \(L-\epsilon\lt \LUB(S)\text{,}\) by definition of the LUB, \(L-\epsilon\) is not an upper bound for \(S\text{,}\) so there is \(t>L-\epsilon\) so that there are infinitely many \(x_n\geq t\text{,}\) which is impossible since there are only finitely many \(x_n>\epsilon\text{.}\) Thus, \(L-\epsilon\geq \LUB(S)=L\text{,}\) which is a contradiction. This proves the claim, and thus \(L\) is a limit point.
