Example 15.4.4.

Let's revisit the “banana” problem. Each arrangement of “banana” corresponds to an ordered partition \((B,A,N)\) of the set \(\{1,2, \dots ,6\}\) with \(|B|=1,|A|=3,|N|=2\text{,}\) where \(i\in B\) if the \(i\)th letter in the rearrangement is a “b”, \(i\in A\) if the \(i\)th letter is an “a”, and \(i\in N\) if the \(i\)th letter is an \(N\text{.}\) Thus, the total number of rearrangements is again

\begin{equation*} {6 \choose 3 , 2, 1} = \frac{6!}{3!2!1!}=60\text{.} \end{equation*}
