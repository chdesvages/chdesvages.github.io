Hence, applying the Fundamental Theorem of Arithmetic to \(x^2\text{,}\) we have \(m=n\text{,}\) \(p_i=q_i\text{,}\) for each \(i\text{,}\) and moreover,

\begin{equation*} 2k_i=3l_i\text{,} \end{equation*}

for each \(i\text{.}\) Since 2 and 3 are coprime, Proposition 10.5 implies that \(k_i\) is divisible by 3 and \(l_i\) is divisible by 2. Let \(k_i'=\frac{k_i}{3}\text{.}\) Let \(n:= p_1^{k_1'}\cdots p_n^{k_n'}\text{.}\) Then \(x=n^3\) and \(y=n^2\text{,}\) as claimed. That is, if \((x,y)\) is a solution with \(x,y\geq 0\text{,}\) then \((x,y)=(n^3,n^2)\) for some \(n\geq 0\text{.}\) Thus, if \((x,y)\) is any solution, then \((x,y)=(\pm n^3,n^2)\) for some \(n\geq 0\text{,}\) or alternatively, \((x,y)=(n^3,n^2)\) for some \(n\in \mathbb{Z}\text{.}\)
