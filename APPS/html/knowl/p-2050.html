Give both an algebraic argument and a combinatorial argument to show (for \(0 \leq k \leq n\))

\begin{equation*} k\binom{n}{k} = n\binom{n-1}{k-1} \end{equation*}

in two different ways. {( Hint for the combinatorial proof: For a set \(S\) with \(|S|=n\text{,}\) the left side is the same as counting the number of pairs \((a,A)\) where \(a\in A\subseteq S\) and \(|A|=k\text{.}\) Can you show the right side counts the same set in a different way?)}
