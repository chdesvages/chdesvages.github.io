Solution 1.

Note that \(f^{-1}(B)=\{x\in X|f(x)\in B\}\text{,}\) so if \(x\in f^{-1}(B)\text{,}\) \(f(x)\in B\text{,}\) hence \(f(f^{-1}(B))\subseteq B\text{.}\)

If we let \(f=\cos:\mathbb{R}\rightarrow \mathbb{R}\) and \(B=[2,3]\text{,}\) then \(f^{-1}(B)=\emptyset\text{,}\) so \(f(f^{-1}(B))=\emptyset\neq B\text{.}\)
