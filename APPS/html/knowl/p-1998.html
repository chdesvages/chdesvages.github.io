Suppose the inductive hyothesis holds for some \(k\geq 2\) and all \(n\text{.}\) Picking an ordered partition \((A_{1}, \dots,A_{k+1})\) with \(|A_{i}|=r_i\) and \(r_{1}+\cdots + r_{k+1}=n\) is the same as picking an \(r_{k+1}\)-member subset \(A_{k+1}\) of \(S\text{,}\) and, for each such, then picking an ordered partition \(A_1, \dots , A_k\) of the remaining \((n - r_{k+1})\)-member set with \(A_i = r_i\) for \(1 \leq i \leq k\text{.}\) By the Multiplication Principle the number of ways of doing this is

\begin{equation*} { n \choose r_{k+1}} \times {n- r_{k+1} \choose r_1, \dots , r_{k}} \end{equation*}

which, by the inductive hypothesis, is

\begin{equation*} { n \choose r_{k+1}} \times \frac{(n- r_{k+1})!}{ r_1! \dots r_{k}!}= \frac{n!}{(n-r_{k+1})! r_{k+1}!}\frac{(n- r_{k+1})!}{ r_1! \dots r_{k}!}= \frac{n!}{r_{1}!r_{2}!\cdots r_{k+1}!}\text{,} \end{equation*}

completing the inductive step.
