Note that \(f^{-1}(f(A))\) is the set of \(z\in X\) for which \(f(z)\in f(A)\text{,}\) which includes \(A\text{.}\) If we let \(f=\cos:\mathbb{R}\rightarrow \mathbb{R}\) again but let \(A=[0,2\pi]\text{,}\) then \(f(A)=[-1,1]\text{,}\) and \(f^{-1}(f(A))=\mathbb{R}\neq A\text{.}\)
