Note that a set of the form \(\{a,a+1, \dots ,a+j\}\subseteq S\) is uniquely determined by the first and last element of the set, which could be any two numbers, or the same number if \(j=0\text{.}\) Thus, the total number of sets of this form is the same as counting the number of subsets of size \(2\) plus the number of sets of size \(1\text{,}\) and adding them together, hence the total number is

\begin{equation*} {n\choose 2} + n\text{.} \end{equation*}
