Recall that \(S_{n}\) has \(n!\) elements. In particular, this means that if we look at the sequence \(f,f^{2},f^{3},..\text{.,}\) eventually this must repeat on itself (otherwise we would have infinitely many distinct permutations on \(n\) integers, whereas we know there are only \(n!\) many). In other words, if we have a deck of \(n\) cards, and we shuffle the cards according to some permutation \(f\text{,}\) then no matter how random the shuffle is, so long as we use the same shuffle \(f\text{,}\) eventually the cards will return to their original order.
