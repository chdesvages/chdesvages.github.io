Proof

We prove the first statement by induction. Let \(P(n)\) be the statement

\begin{equation*} z^n = r^{n} (\cos (n\theta) + i \sin (n\theta))\text{.} \end{equation*}

The base case \(n=1\) holds immediately by definition.

Assume \(P(n)\) holds for some integer \(n\geq 1\text{.}\)

\begin{equation*} z^{n+1} = zz^n = z\times r^{n} (\cos (n\theta) + i \sin (n\theta)) \end{equation*}

using the induction hypothesis. And so this

\begin{equation*} =r(\cos \theta + i \sin \theta) \times r^{n} (\cos (n\theta) + i \sin (n\theta)) \end{equation*}
\begin{equation*} = r^{n+1} ( \cos \theta\cos (n\theta) - \sin \theta\sin (n\theta) +i(\cos \theta\sin (n\theta) + \sin \theta\cos (n\theta))) \end{equation*}
\begin{equation*} = r^{n+1} (\cos ((n+1)\theta) + i \sin ((n+1)\theta))\text{,} \end{equation*}

which proves \(P(n+1)\) holds.

Since \(P(1)\) and \(P(n)\rightarrow P(n+1)\) it follows that the first equation in the theorem holds by the principle of mathematical induction.

For the second part, we just note that by (10.1.1),

\begin{equation*} \frac{1}{\cos n \theta +i\sin n \theta} =\cos n \theta - i\sin n \theta\text{,} \end{equation*}

and so

\begin{equation*} z^{-n}=(z^{n})^{-1} = (r^{n}(\cos n\theta + i\sin n\theta))^{-1} = r^{-n} (\cos n \theta - i\sin n \theta) = r^{-n} e^{-i n\theta}\text{.} \end{equation*}
