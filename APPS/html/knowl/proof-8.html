Proof

For \(y\in f(X)\text{,}\) \(f^{-1}(y)\) is an equivalence class: since \(y\in f(X)\text{,}\) \(y=f(x)\) for some \(x\in X\text{,}\) and if \([x]\) denotes the equivalence class for \(x\text{,}\)

\begin{equation*} [x]=\{z\in X:x\sim z\} = \{z\in X:f(z)=f(x)\} = \{z\in X:f(z)=y\} =f^{-1}(\{y\})\text{.} \end{equation*}

hence \(F\) is a function from \(f(X)\) to \(S\text{.}\) We need to show that it is injective and surjective:

  • (Injectivity) Let \(y,z\in f(X)\) and suppose \(F(y)=F(z)\text{,}\) we will show that \(y=z\text{.}\) Since \(F(y)=F(z)\text{,}\) this means \(f^{-1}(y)=f^{-1}(z)\text{.}\) Let \(x\in f^{-1}(y)\text{,}\) so \(f(x)=y\text{.}\) But then \(x\in f^{-1}(y)=f^{-1}(z)\text{,}\) so \(f(x)=z\text{,}\) thus \(y=f(x)=z\text{.}\) This proves injectivity.

  • (Surjectivity) Let \([x]\in S\) be an equivalence class. Let \(y=f(x)\text{.}\) Then \([x]=f^{-1}(y)\) and \(y\in f(X)\text{.}\) Thus, \(F(y)=f^{-1}(y)=[x]\text{.}\) This implies surjectivity.
