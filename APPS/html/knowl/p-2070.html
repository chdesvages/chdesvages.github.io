Here we will introduce some useful notation for denoting permutations. To represent a permutation \(f\in S_n\text{,}\) we just need to remember where each integer \(k\in \{1,2,...,n\}\) gets sent to under \(f\) and then write out a chart using a matrix:

\begin{equation*} \left(\begin{matrix}1 \amp 2 \amp 3 \amp \cdots \amp n \crf(1) \amp f(2) \amp f(3) \amp \cdots \amp f(n) \cr \end{matrix} \right) \end{equation*}

where the first row represents the domain of \(f\) and the bottom row represents where each value is sent to under \(f\text{.}\) For example,

\begin{equation*} \left(\begin{matrix}1 \amp 2 \amp 3 \\ 3 \amp 1 \amp 2 \end{matrix} \right) \end{equation*}

is the permutation in \(S_{3}\) so that \(f(1)=3\text{,}\) \(f(2)=1\text{,}\) and \(f(3)=2\text{.}\)
