Definition 14.0.1.

We say two sets \(A\) and \(B\) have the same cardinality if there is a bijective function \(f:A\rightarrow B\text{,}\) and we write \(|A|=|B|\) or \(A\sim B\text{.}\) If there is an injective man \(f:A\rightarrow B\text{,}\) we write \(|A|\leq |B|\text{,}\) and if \(|A|\leq |B|\) and there is no injective map from \(B\) to \(A\text{,}\) we write \(|A|\lt |B|\text{.}\)

When \(A\sim \mathbb{N}\text{,}\) we say \(A\) is countable.
