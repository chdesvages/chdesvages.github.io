Since \(n\) is not prime, \(n=ab\) for some integers \(1\lt a,b\lt n\text{.}\)

  1. If \(a\lt b\text{,}\) then

    \begin{equation*} (n-1)!=1\cdot 2 \cdots a\cdots b\cdots (n-1)\text{.} \end{equation*}

    so \(n=ab\) divides \((n-1)!\text{.}\)

  2. if \(a=b\text{,}\) then \(n=a^2\) and \(a>2\text{,}\) so \(2a\leq a^2-1\text{,}\) and

    \begin{equation*} (n-1)!=1\cdot 2 \cdots a\cdots 2a\cdots (a^2-1) \end{equation*}

    so we see that \(a^2\) divides \((n-1)!\text{.}\)
