The binomial coefficient \({n\choose k}\) counts the number of ways to pick a subset of size \(k\) from a set of size \(n\text{,}\) or in other words, the number of ways of splitting a set of size \(n\) into a set of size \(k\) and a set of size \(n-k\text{.}\) What if we wanted to count the number of ways to partition a set into more than two sets?
