Definition 6.2.1.

Given a nonempty set \(A\subseteq \mathbb{R}\text{,}\) a number \(L\) is a least upper bound (LUB) or supremum for \(A\) if

  1. \(L\) is an upper bound for \(A\text{,}\) that is, \(x\leq L\) for all \(x\in A\text{,}\) and

  2. for all \(t\lt L\text{,}\) \(t\) is not an upper bound, that is, there is \(x\in A\) so that \(x>t\text{.}\)

Similarly, we say \(L\) is a greatest lower bound (GLB) or infimum for \(A\) if

  1. \(L\) is a lower bound for \(A\text{,}\) that is, \(x\geq L\) for all \(x\in A\text{,}\) and

  2. for all \(t>L\text{,}\) \(t\) is not a lower bound, that is, there is \(x\in A\) so that \(x\lt t\text{.}\)

If an LUB exists for a set \(A\text{,}\) we denote it by \(\LUB(A)\text{.}\) Similarly, we denote a GLB for \(A\) by \(\GLB(A)\text{.}\)
