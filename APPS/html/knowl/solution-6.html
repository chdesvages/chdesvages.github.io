Solution 1.15.1.1.

  1. Recall that \(A\Rightarrow B\) is equivalent to \(\bar{A}\) or \(B\text{,}\) so we just need to find a statement so that \(A\) is false (so \(\bar{A}\) is true) and \(B\) is true. This could be \(A\)= “2 is odd” and \(B\)= “2 is even.”

  2. Recall that \(A\Rightarrow B\) is equivalent to \(\bar{A}\) or \(B\text{,}\) so if \(B\) is false, then \(\bar{A}\) must be true (i.e. \(A\) is false). We can let \(A=B\)= “2 is odd”, for example.
