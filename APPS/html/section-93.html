Section 16.8 Order of a permutation

Recall that \(S_{n}\) has \(n!\) elements. In particular, this means that if we look at the sequence \(f,f^{2},f^{3},..\text{.,}\) eventually this must repeat on itself (otherwise we would have infinitely many distinct permutations on \(n\) integers, whereas we know there are only \(n!\) many). In other words, if we have a deck of \(n\) cards, and we shuffle the cards according to some permutation \(f\text{,}\) then no matter how random the shuffle is, so long as we use the same shuffle \(f\text{,}\) eventually the cards will return to their original order.

In this section we'll show how to compute how soon powers of a permutation repeat.

Definition 16.8.1.

Let \(f\) be a permutation in \(\mathrm{S}_n\text{.}\) The smallest positive integer \(m\) such that

\begin{equation*} f^m=\iota \end{equation*}

is called the order of the permutation \(f\text{.}\)

The order of a cycle of length \(r\) is just \(r\text{,}\) which we saw earlier. In a few steps we will show how to compute the order of a permutation in general.

Lemma 16.8.2.

Let \(f\in\mathrm{S}_n\) have order \(m\text{.}\) If \(f^k=\iota\) for some integer \(k\in\mathbb{N}\text{,}\) then \(m\) divides \(k\text{.}\)

Proof.

If \(m\) is the order of \(f\in\mathrm{S}_n\) and \(f^k=\iota\text{,}\) then by definition of \(m\text{,}\) \(m\leq k\text{.}\) By the remainder theorem, there are \(q\geq 0\) and \(0\leq r\lt m\) so that \(k=qm+r\text{,}\) and so

\begin{equation*} f^{k} = f^{qm+r} = (f^{m})^{q} f^{r} = (\iota)^{q}f^{r} = \iota f^{r}= f^{r}\text{.} \end{equation*}

Recall that \(m\) is the smallest positive integer for which \(f^{m}=\iota\text{,}\) but now we have \(r\in \{0,1,...,m-1\}\) (and so \(r\lt m\)) for which \(f^{r}=\iota\text{.}\) This is only possible if \(r\) is not positive, i.e. if \(r=0\text{.}\) Thus, \(k=qm+0=qm\) so \(m|k\text{.}\)

Lemma 16.8.3.

Let \(f\in\mathrm{S}_n\) and let \(m\) be its order. Write

\begin{equation*} f=\sigma_{1}\sigma_{2}\cdots\sigma_{s} \end{equation*}

where \(\sigma_{1},\ldots,\sigma_{s}\) are disjoint cycles of lengths \(r_{1},\ldots,r_{s}\) respectively. Then

\begin{equation*} m=\mathrm{lcm}\big(r_{1},r_{2},\ldots,r_{s}\big)\text{,} \end{equation*}

where \mathrm{lcm} stands for the least common multiple, that is, the smallest positive integer that is divisible by \(\sigma_{i}\) for each \(i=1,...,s\text{.}\)

Proof.

Recall that the order of a cycle is just its length, so we know that \(\sigma_i^{r_i}=\iota\text{.}\) Notice that if \(d= \mathrm{lcm}\big(r_{1},r_{2},\ldots,r_{s}\big)\text{,}\) then \(r_i|d\) for all \(i\text{.}\) Also recall that disjoint cycles commute, so we can split the following product:

\begin{equation*} f^{d} = (\sigma_{1}\sigma_{2}\cdots\sigma_{s})^{d} =\sigma_{1}^{d}\sigma_{2}^{d}\cdots \sigma_{s}^{d} =(\sigma_{1}^{r_{1}})^{d/r_{1}}(\sigma_{2}^{r_{2}})^{d/r_{2}}\cdots(\sigma_{s}^{r_{s}})^{d/r_{s}} = (\iota)^{d/r_{1}}(\iota)^{d/r_{2}}\cdots (\iota)^{d/r_{s}}=\iota\text{.} \end{equation*}

By the previous lemma, this means \(m|d\text{.}\) However, note that since \(m\) is the order of \(f\text{,}\)

\begin{equation*} \iota = f^{m} = (\sigma_{1}\sigma_{2}\cdots\sigma_{s})^{m} =\sigma_{1}^{m}\sigma_{2}^{m}\cdots \sigma_{s}^{m} \end{equation*}

and since these cycles are disjoint, we must have that \(\sigma_{i}^{m}=\iota\) for each \(i\text{.}\) By the previous lemma again, this means \(r_i|m\) (since \(r_i\) is the order of \(\sigma_i\)). By the definition of the least common multiple, this means \(d|m\text{.}\) Finally, \(m|d\) and \(d|m\) (and both being positive integers) imply \(d=m\text{.}\)

Using this Lemma, you can now always compute \(m\) by finding the cycle decomposition.

Example 16.8.4.

The order of the permutation

\begin{equation*} \left(\begin{matrix}1 \amp 2 \amp 3 \amp 4 \amp 5 \amp 6 \amp 7 \amp 8\cr3 \amp 4 \amp 6 \amp 1 \amp 5 \amp 2 \amp 8 \amp 7\cr \end{matrix} \right) = \big(1\ 3\ 6\ 2\ 4\big)\big(7\ 8\big)\in\mathrm{S}_{8}\text{,} \end{equation*}

is \(\mathrm{lcm}(2, 5, 1)=\mathrm{lcm}(2, 5)=10\text{,}\) while the order of

\begin{equation*} \left(\begin{matrix}1 \amp 2 \amp 3 \amp 4 \amp 5 \amp 6 \amp 7 \amp 8\cr5 \amp 6 \amp 7 \amp 8 \amp 1 \amp 2 \amp 3 \amp 4\cr \end{matrix} \right) = (1 \; 5)(2 \; 6)(3\; 7)(4\; 8) \in\mathrm{S}_{8}\text{,} \end{equation*}

is \(\mathrm{lcm}(2,2,2,2)=2\text{.}\)
