Section 6.1 Upper and Lower Bounds

Definition 6.1.1.

Let \(A\) be a nonempty subset of \(\R\text{.}\)

  • \(A\) is said to be bounded above if there exists \(M\in\R\) such that \(x\leq M\) for all \(x\in A\text{.}\) Such a number \(M\) is called an upper bound for \(A\text{.}\) If no upper bound exists, we say \(A\) is unbounded above.

  • Similarly, \(A\) is said to be bounded below if there exists \(m\in\R\) such that \(x\geq m\) for all \(x\in A\text{.}\) Such a number \(m\) is called a lower bound for \(A\text{.}\) If no lower bound exists, we say \(A\) is unbounded below.

  • We say \(A\) is bounded if it is both bounded above and bounded below.

Checkpoint 6.1.2.

Show that \(A\) is bounded if and only if there is \(t\geq 0\) so that \(|x|\leq t\) for all \(x\in A\text{.}\)

Solution.

If \(A\) is bounded, there are \(m,M\in\mathbb{R}\) so that \(m\leq x\leq M\) for all \(x\in A\text{.}\) We claim that \(|x|\leq \max\{|m|,|M|\}\) for all \(x\in A\text{.}\) Indeed, if \(x\in A\) then

\begin{equation*} x\leq M\leq |M| \leq \max\{|m|,|M|\}\text{,} \end{equation*}

and similarly

\begin{equation*} x\geq m \geq - |m| \geq -\max\{|m|,|M|\} \end{equation*}

so that

\begin{equation*} |x|\leq \max\{|m|,|M|\} \end{equation*}

for all \(x\in A\text{.}\) Conversely, if \(|x|\leq t\) for all \(x\in A\text{,}\) then \(-t\leq x\leq t\) for all \(x\in A\) (since either \(x\geq 0\) in which case \(x=|x|\leq t\text{,}\) or \(x\leq 0\text{,}\) so \(x=-|x|\geq -t\)). Thus, \(-t\) is a lower bound and \(t\) is an upper bound for \(A\text{.}\)

Recall that we write intervals using the notation

\begin{equation*} [a,b]=\{x\in\mathbb{R} \;\; | \;\; a\leq x\leq b\}, \;\;\; (a,b)=\{x\in\mathbb{R} \;\; | \;\; a\lt x\lt b\}\text{,} \end{equation*}
\begin{equation*} (a,b]=\{x\in\mathbb{R} \;\; | \;\; a\lt x\leq b\}, \;\;\; [a,b)=\{x\in\mathbb{R} \;\; | \;\; a\leq x\lt b\}\text{.} \end{equation*}

For unbounded intervals, we write

\begin{equation*} [a,\infty)=\{x\in\mathbb{R} \;\; | \;\; a\leq x \}, \;\;\; (a,\infty)=\{x\in\mathbb{R} \;\; | \;\; a\lt x \}\text{,} \end{equation*}
\begin{equation*} (-\infty,b]=\{x\in\mathbb{R} \;\; | \;\; x\leq b\}, \;\;\; (-\infty,b)=\{x\in\mathbb{R} \;\; | \;\; x\lt b\}\text{.} \end{equation*}
Example 6.1.3.

  1. Let \(A=[0,3)\text{.}\) \(A\) is bounded above since \(3\) is an upper bound; \(A\) is bounded below since \(0\) is a lower bound.

  2. Let \(A = \{ t+\frac{1}{t}\, :\, t>0\}\text{.}\) \(A\) is bounded below since \(0\) is a lower bound. (We could have also shown it was bounded below by using the AM-GM inequality: for \(t>0\text{,}\)

    \begin{equation*} \frac{t+1/t}{2}\geq \sqrt{t\cdot \frac{1}{t}}=1\text{,} \end{equation*}

    so \(2\) is also a lower bound for \(A\text{.}\)) But \(A\) is not bounded above, since, for any \(M\in\mathbb{R}\) we can find \(t>0\) such that \(t+\frac{1}{t}\geq M\text{:}\) if \(M \leq 0\) then any \(t>0\) works; if \(M>0\) then \(t= M\) works since \(M+\frac{1}{M}\in A\) is greater than \(M\text{.}\)

  3. Let \(A=\{ (-1)^n(1-\frac{1}{n})\, |\, n\in\N\}=\{0,\frac{1}{2},-\frac{2}{3},\frac{3}{4},-\frac{4}{5},\frac{5}{6},\cdots \}\text{.}\) \(A\) is bounded: 1 is an upper bound since for all \(n\text{:}\)

    \begin{equation*} (-1)^{n}\left(1-\frac{1}{n}\right)\leq 1-\frac{1}{n}\lt 1 \end{equation*}

    and one can show similarly \(-1\) is a lower bound.

  4. Let \(A=\{\sin n\, |\, n\in\N\}=\{ \sin 1,\, \sin 2,\, \sin 3,\, \cdots \}\text{.}\) Because \(-1 \leq \sin x \leq 1\) for all \(x\text{,}\) \(A \subseteq [-1,1]\text{,}\) so \(A\) is bounded.
