Section 6.3 Exercises

The exercises in Liebeck's book relevant to this section are in Chapter 22.

Checkpoint 6.3.1.

Let \(A,B\subseteq \mathbb{R}\) be nonempty sets that are bounded above. For each statement below, either prove it or provide a counterexample.

  1. If \(A\) is bounded above, then \(\LUB(-A)=-\LUB(A)\text{,}\) where \(-A=\{-x \; | \; x\in A\}\text{.}\) Solution.

    This is not true: if \(A=(-1,2)\text{,}\) then \(-A=(-2,1)\text{,}\) so \(-\LUB(A)=-2\neq \LUB(-A)=-1\text{.}\) Another example is \(A = (-\infty, 0)\) with \(\LUB(A) = 0\text{,}\) for which \(-A\) is not even bounded above, and thus \(-A\) has no LUB. Many more examples are possible.

  2. If \(A\) is bounded above, then \(-A\) is bounded below and \(\GLB(-A)=-\LUB(A)\text{.}\) Solution.

    This is true: Let \(L=\LUB(A)\text{,}\) we claim \(\GLB(-A)=-L\text{.}\) First we show it is a lower bound: if \(x\in -A\text{,}\) then \(-x\in A\text{,}\) so \(-x\leq L\text{,}\) hence \(x\geq -L\text{.}\) Now we show it is the greatest lower bound: let \(t>-L\text{,}\) we will show there is \(x\in -A\) with \(x\lt t\text{.}\) Note that \(-t\lt L\text{,}\) and since \(L=\LUB(A)\text{,}\) there is \(y\in A\) with \(y>-t\text{,}\) so \(-y\lt t\) and \(-y\in -A\text{,}\) thus \(t\) is not a lower bound. This proves \(\GLB(-A)=-L\text{.}\)

  3. If \(c\in \mathbb{R}\text{,}\) then \(\LUB(c+A)=c+\LUB(A)\) where \(c+A=\{c+x \; | \; x\in A\}\text{.}\) Solution.

    This is true: let \(L=\LUB(A)\text{,}\) we claim \(\LUB(c+A)=c+L\text{.}\) First we show \(c+L\) is an upper bound: if \(x\in c+A\text{,}\) then \(x-c\in A\text{,}\) so \(x-c\leq L\text{,}\) so \(x\leq c+L\text{.}\) Now we show that it is a least upper bound, i.e., that for \(t\lt c+L\text{,}\) there is \(x\in c+A\) with \(x>t\text{:}\) Let \(t\lt c+L\text{,}\) then \(t-c\lt L\text{,}\) and since \(L=\LUB(A)\text{,}\) there is \(y\in A\) with \(y>t-c\text{,}\) and so \(y+c>t\) and \(y+c\in A\text{,}\) so \(t\) is not an upper bound for \(A\text{.}\)

  4. \(\LUB(A)=\max(A)\text{.}\) Solution.

    This is not true in general since \(\max(A)\) may not be defined. A concrete conterexample is \(A = (0,1)\) for which \(\LUB(A) =1\text{,}\) but which has no maximum. Many more examples are possible.

  5. If \(c>0\text{,}\) then \(\LUB(cA)=c\LUB(A)\) where \(cA=\{cx\; | \; x\in A\}\text{.}\) Solution.

    This is true, its proof is very similar to (c), so we omit it.

  6. \(\LUB(AB)=\LUB(A)\cdot \LUB(B)\) where \(AB=\{xy \;\; | \;\; x\in A,\; y\in B\}\text{.}\) Solution.

    Not true: let \(A=(-2,1)\) and \(B=(-2,1)\text{,}\) then \(AB=(-2,4)\) so \(\LUB(AB)=4\neq \LUB(A)\cdot \LUB(B)=1\text{.}\) Many more examples are possible.

  7. if \(A\subseteq B\text{,}\) then \(\LUB(A)\leq \LUB(B)\text{.}\) Solution.

    This is true: note that \(x\leq \LUB(B)\) for all \(x\in B\text{,}\) and so \(x\leq \LUB(B)\) for all \(x\in A\) since \(A\subseteq B\text{.}\) Hence, \(\LUB(A)\leq \LUB(B)\text{.}\)

Checkpoint 6.3.2.

Suppose \(A\) and \(B\) are bounded above. Show that \(\LUB(A+B)=\LUB(A)+\LUB(B)\text{,}\) where \(A+B=\{x+y\; | \; x\in A,\; y\in B\}\text{.}\)

Solution.

Let \(L=\LUB(A)\) and \(L'=\LUB(B)\text{,}\) we'll show \(\LUB(A+B)=L+L'\text{.}\) First we show (a) that \(L+L'\) is an upper bound for \(A+B\text{:}\) Note that if \(z\in A+B\text{,}\) then \(z=x+y\) where \(x\in A\) and \(y\in B\text{,}\) so

\begin{equation*} z=x+y\leq L+L' \end{equation*}

and so \(L+L'\) is an upper bound. Now we show (b) that if \(t\lt L+L'\text{,}\) then there is \(z\in A+B\) with \(z>t\text{.}\) Recall that since \(L=\LUB(A)\text{,}\) for any \(s\lt L\) there is \(x\in A\) with \(x>s\text{,}\) and since \(L'=\LUB(B)\text{,}\) for any \(s'\lt L'\) there is \(y\in B\) with \(y>s'\text{,}\) and so \(x+y>s+s'\) and \(x+y\in A+B\text{.}\) Hence, we just need to find values of \(s\lt L\) and \(s'\lt L'\) so that \(s+s= t\text{,}\) since then \(x+y>s+s'= t\text{.}\) Let's rewrite \(s\) and \(s'\) as \(s=L-r\) and \(s'=L-'r'\) where \(r,r'>0\text{.}\) Then we want to find \(r,r'>0\) so that \(t=s+s'=L+L'-r-r'\text{,}\) i.e. so that \(r+r'=L+L'-t\text{,}\) so we can just choose \(r=r'=\frac{L+L'-t}{2}\text{.}\)

Checkpoint 6.3.3.

Determine which of the following sets are bounded or unbounded above.

  1. \(A=\{x \in \R\; | \; x^2-3x+2\lt 2\}\text{.}\) Solution.

    If \(x\) is in \(A\text{,}\) then

    \begin{equation*} x^2-3x=x(x-3)\lt 0 \end{equation*}

    so exactly one of \(x\) or \(x-3\) is negative, which is only possible if \(x-3\lt 0\lt x\text{.}\) Thus for all \(x \in A\text{,}\) \(0\lt x\lt 3\text{,}\) so \(0\) and \(3\) are lower and upper bounds for \(A\) respectively. Therefore \(A\) is bounded.

  2. \(A= \{x \in \R \; | \; \cos x\lt 1/2\}\text{.}\) [You may assume any familiar properties of the cosine function.] Solution.

    Note that \(\cos \left(\frac{\pi}{2}+2\pi n\right)=0\) for all \(n\in\mathbb{Z}\) and so \(\frac{\pi}{2}+2\pi n\) is in \(A\) for all \(n\text{.}\) In particular, \(A\) is unbounded above and below: if \(M\in\mathbb{R}\text{,}\) by the Archimedean property, we can find \(n\) so that \(2\pi n>M-\frac{\pi}{2}\text{,}\) i.e. so that \(2\pi n+\frac{\pi}{2}>M\text{,}\) hence \(A\) is not bounded above. That it is not bounded below is similar.

  3. \(A = \{n^2-n \; | \; n\in\mathbb{N}\}\text{.}\) Solution.

    If \(n\in\mathbb{N}\text{,}\) then \(n\geq 1\) and so \(n^2\geq n\text{,}\) i.e. \(n^2-n \geq 0\text{,}\) hence \(A\) is bounded below with \(0\) being a lower bound. It is not bounded above. Indeed, let \(M \in \R\text{.}\) Note that for \(n > 1\) we have \(n^2-n=n(n-1)> n\text{,}\) so by the Archimedean property, we may find \(n>\max\{M,1\}\) such that \(n^2-n=n(n-1)\geq n>M\text{.}\) Thus \(M\) cannot be an upper bound for \(A\text{,}\) and we deduce that \(A\) is not bounded above.

  4. \(A=\{ab \;\;| \;\; a, b \in \R, \; a+b=1\}\text{.}\) Solution.

    Note that if \(a+b=1\text{,}\) then \(b=1-a\text{,}\) so that

    \begin{equation*} A = \{a(1-a)\;\; |\;\;a\in\mathbb{R}\} = \{- a^2 +a \;\; |\;\;a\in\mathbb{R}\}\text{.} \end{equation*}

    Which real numbers \(r\) are members of this set? It is precisely those \(r \in \R\) such that the quadratic equation

    \begin{equation*} -a^2 + a =r \end{equation*}

    has a real solution. To figure this out we complete the square:

    \begin{equation*} -a^2 + a =r \iff -\left(a-\frac{1}{2}\right)^2 = r -\frac{1}{4} \iff \left(a-\frac{1}{2}\right)^2 =\frac{1-4r}{4}\text{,} \end{equation*}

    and so long as \((1-4r)/4 \geq 0\) — that is, \(r \leq 1/4\) — it has a real square root by Proposition 2.7. Thus \(r \in A\) if and only if \(r \leq 1/4\text{.}\) Therefore

    \begin{equation*} A = (- \infty, 1/4] \end{equation*}

    and this is bounded above (with \(1/4\) as an upper bound) but not bounded below.

Checkpoint 6.3.4.

Find the least upper bounds for the following sets:

  1. \(\{\frac{1}{n} \; | \; n\in \mathbb{N}\}\text{.}\) Solution.

    Note that for \(n\geq 1\text{,}\) \(\frac{1}{n}\leq 1\text{,}\) so \(1\) is an upper bound. Since \(1\) is in the set, it is in fact a maximum and thus equal to the least upper bound.

  2. \(\{-\frac{1}{n^2} \; | \; n\in\mathbb{N}\}\text{.}\) Solution.

    We claim that the LUB is \(0\text{.}\) It is clearly an upper bound since all elements of the set are negative. Now we must show it is the least upper bound: let \(t\lt 0\text{,}\) we'll show there is \(-\frac{1}{n^2}>t\text{.}\) This is the same as finding \(n\) so that \(n>(-t)^{-1/2}\text{,}\) which is possible by the Archimedean property.

  3. \(\{2-x^2 \; | \; x\in \R\setminus\{0\}\}\text{.}\) Solution.

    We claim the least upper bound is 2. Since \(x^2\geq 0\text{,}\) \(2-x^2\leq 2\) for any \(x\neq 0\text{,}\) so \(2\) is an upper bound. Now suppose \(t\lt 2\text{,}\) we'll show there is \(x\) so that \(2-x^2>t\text{,}\) i..e. so that \(x^2\lt 2-t\text{.}\) Since \(2-t>0\text{,}\) we can just set \(x=\sqrt{\frac{2-t}{2}}\) and this will work. This proves \(2\) is the least upper bound.

  4. \(\{x\in\mathbb{Q}: x^2\lt 2\}\text{.}\) Solution.

    This is the same as the set of \(x\in\mathbb{Q}\) so that

    \begin{equation*} 0>x^2-2=(x-\sqrt{2})(x+\sqrt{2}) \end{equation*}

    so exactly one of these terms must be negative, so we must have \(-\sqrt{2}\lt x\lt \sqrt{2}\text{,}\) that is,

    \begin{equation*} \{x\in\mathbb{Q}: x^2\lt 2\}=(-\sqrt{2},\sqrt{2}) \end{equation*}

    We claim the least upper bound is \(\sqrt{2}\text{:}\) clearly \(\sqrt{2}\) is an upper bound, so now we must show it is a least upper bound: let \(t\lt \sqrt{2}\text{,}\) we'll show there is \(x\) in the set with \(x>\sqrt{2}\text{.}\) From Week 1, we saw that for any \(a\lt b\text{,}\) there is a rational \(x\) with \(a\lt x\lt b\text{,}\) so in particular, there is \(x\) with \(t\lt x\lt \sqrt{2}\text{.}\) This completes the proof that \(\sqrt{2}\) is a LUB. The proof that \(-\sqrt{2}\) is the GLB is similar.
